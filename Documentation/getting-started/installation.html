Installation Guide

How to install and set up Beep Controls in your .NET project

Prerequisites

  • .NET 6, 7, 8, or 9 SDK installed
  • Visual Studio 2022 (17.0+) or VS Code with C# extension
  • A WinForms project targeting one of the supported .NET versions

Install via NuGet

The easiest way to install Beep Controls is through the NuGet Package Manager.

Package Manager Console

Install-Package TheTechIdea.Beep.Winform.Controls

.NET CLI

dotnet add package TheTechIdea.Beep.Winform.Controls

PackageReference (csproj)

<PackageReference Include="TheTechIdea.Beep.Winform.Controls" Version="1.0.*" />

Install from Source

If you prefer to build from source:

Clone and build

git clone https://github.com/The-Tech-Idea/Beep.Winform.git
cd Beep.Winform
dotnet build WinFormsApp.sln

Then add a project reference in your application:

Project reference

<ProjectReference Include="..\path\to\TheTechIdea.Beep.Winform.Controls\TheTechIdea.Beep.Winform.Controls.csproj" />

Verify Installation

After installation, verify by using BeepiFormPro as your main form:

Verify installation

using TheTechIdea.Beep.Winform.Controls;
using TheTechIdea.Beep.Winform.Controls.Forms.ModernForm;

public class MainForm : BeepiFormPro
{
    public MainForm()
    {
        Text = "Verify Install";
        Size = new Size(400, 300);
        FormStyle = FormStyle.Material;

        var btn = new BeepButton
        {
            Text = "Hello Beep!",
            Location = new Point(20, 80),
            Size = new Size(150, 40)
        };
        Controls.Add(btn);
    }
}

If the form renders with a Material-styled caption bar and a themed button, the library is working correctly.
