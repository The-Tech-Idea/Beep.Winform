BeepMetricWidget

KPI and metric display widget with six visual styles including trend indicators, progress bars, circular gauges, comparison panels, and card layouts.

Stable v1.0.164 Beep Widgets / Data / KPI

Overview

BeepMetricWidget is a paint-based WinForms widget that inherits from BaseControl. It uses an IWidgetPainter pipeline to render rich visuals without child controls. The Style property selects one of 6 built-in render modes; changing it at runtime immediately swaps the painter and triggers a repaint.

Key Features

📊 Six Visual Modes

From a single plain number to a full card layout with icon — one property switch.

📈 Trend Indicators

Arrow icons with positive/negative colours and percentage labels.

⭕ Circular Gauge

Smooth arc gauge drawn entirely in GDI+ — no external charting library needed.

🎨 Theme-Aware Colors

AccentColor, SuccessColor, and ErrorColor default to the active BeepTheme.

🔔 Click Events

Separate events for value area, trend area, and icon clicks.

📐 Compact Default

200×120 default — fits in tight dashboard grids.

Basic Usage

var metric = new BeepMetricWidget
{
    Style         = MetricWidgetStyle.ValueWithTrend,
    Title         = "Monthly Revenue",
    Value         = "$127,450",
    TrendValue    = "+8.3%",
    TrendDirection = "up",
    ShowTrend     = true,
    Size          = new Size(220, 120)
};
this.Controls.Add(metric);

Event Handling

metric.ValueClicked += (s, e) =>
{
    // e.Data contains the current Value string
    ShowRevenueBreakdown(metric.Value);
};

metric.TrendClicked += (s, e) =>
{
    ShowTrendHistory();
};

Best Practices

  • Set Style before adding the widget to the form to avoid double repaints.
  • Adjust Size to fit your layout — defaults are minimum comfortable sizes.
  • Read BeepEventDataArgs.Data in event handlers for context-specific click data.
  • Call Invalidate() after updating data properties to trigger a repaint.
  • Let theme colors flow from BeepStyling.CurrentTheme — avoid overriding color properties unless customization is explicitly desired.
