Overview

Namespace: TheTechIdea.Beep.Winform.Controls.Charts
Assembly: TheTechIdea.Beep.Winform.Controls.dll
Inheritance: BaseControl

BeepChart is a full-featured charting control supporting 5 chart types (Bar, Line, Pie, Area, Bubble) with interactive viewport control (zoom/pan/auto-scale), performance optimization for datasets exceeding 10,000 points, real-time streaming, trackball with crosshair and multi-series tooltips, and 18 events with dedicated EventArgs classes. It is decomposed across 5 partial files with ~20 helper classes organized into five subsystems.

Painting Pipeline

DrawContent(Graphics g)
  -> Paint(Graphics g, Rectangle bounds)
       -> IChartPainter.Initialize(owner, theme)
       -> IChartPainter.AdjustLayout(drawingRect, ctx) -> ChartLayout
            |-- IChartAxisPainter.AdjustPlotRect(g, axisCtx) -> AxisLayout (48px margins)
       -> IChartPainter.DrawBackground(g, ctx)
            |-- CardChartPainter: rounded card background + border
            |-- IChartAxisPainter.DrawAxes(g, axisCtx) -> X/Y axis lines + titles
            |-- IChartAxisPainter.DrawTicks(g, axisCtx) -> numeric | date | text
       -> [foreach DataSeries, palette index]:
            IChartSeriesPainter.DrawSeries(g, plotRect, data, toX, toY,

       -> IChartLegendPainter.DrawLegend(g, chartRect, series, palette,
            font, textColor, backColor, shapeColor, owner, onToggleSeries, ...)
       -> IChartPainter.DrawForeground(g, ctx) -> title, subtitle, value
       -> IChartPainter.UpdateHitAreas(owner, ctx, notifyAreaHit)

Painter Contracts (4 Interfaces)

IChartPainter

Top-level chart painter. Handles layout adjustment, background/foreground, and hit area registration. Concrete implementation: CardChartPainter renders charts as cards with rounded corners, title bars, and value text areas.

internal interface IChartPainter
{
    void Initialize(BaseControl owner, IBeepTheme theme);
    ChartLayout AdjustLayout(Rectangle drawingRect, ChartLayout ctx);
    void DrawBackground(Graphics g, ChartLayout ctx);
    void DrawForeground(Graphics g, ChartLayout ctx);
    void UpdateHitAreas(BaseControl owner, ChartLayout ctx,
        Action<string, Rectangle> notifyAreaHit);
}

// ChartLayout data class:
// DrawingRect, PlotRect, Radius, AccentColor, TitleBottom

IChartSeriesPainter

Responsible for rendering individual chart series (bars, lines, pie slices, etc.). Five concrete implementations returned by SeriesPainterFactory.

IChartAxisPainter

Renders coordinate axes with intelligent tick calculation. Concrete implementation: CartesianAxisPainter (internal sealed). AxisLayout data class carries Bounds, PlotRect, BottomAxisType, LeftAxisType, XMin/XMax/YMin/YMax, XTitle/YTitle, XLabelAngle/YLabelAngle, AxisColor, GridLineColor, and category/date epoch data.

IChartLegendPainter

Renders a color-coded series reference panel. Concrete implementation: RightSideLegendPainter. Supports series toggle via onToggleSeries(int) callback and LegendPlacement (Right, Bottom, Top, Left). EnableLegendIsolate enables single-series isolation mode.

Series Painters (5 Types)

SeriesPainterFactory.Get(ChartType) resolves the chart type to its painter. LineSeriesPainter is the default fallback.

ChartTypePainterRendering Style
BarBarSeriesPainterVertical bars with configurable width (BarWidth=0.8) and padding (BarPadding=0.2). Supports stacked mode and data labels on top. Negative values extend downward.
LineLineSeriesPainterConnected line segments with optional cubic Bezier smoothing (SmoothLines). Supports point markers: Circle, Square, Diamond, Triangle at configurable size (default 6px).
PiePieSeriesPainterProportional pie/donut slices. Donut mode when InnerRadius > 0. Exploded slice on hover. Hit areas are pie-slice-shaped polygons.
AreaAreaSeriesPainterFilled polygon below each line to baseline. Semi-transparent (FillOpacity=0.3). Stacked mode accumulates vertically.
BubbleBubbleSeriesPainterProportional circles at XY coordinates. Circle area (not radius) proportional to Size value.

Axis & Legend System

CartesianAxisPainter

Implements IChartAxisPainter. Three-phase rendering:

  • AdjustPlotRect -- Adds 48px margins on all sides of the chart area
  • DrawAxes -- Renders X/Y axis lines and titles (titles drawn rotated)
  • DrawTicks -- Dispatches to three sub-renderers based on axis type:
    • NumericTicks -- "Nice" step sizes: 1, 2, 5, 10, 20, 50, 100, etc. No unnecessary decimal places.
    • DateTicks -- Auto-granularity: < 1h -> minutes, < 24h -> hours, < 30d -> days, < 365d -> months, >= 1y -> years
    • TextTicks -- Category-indexed labels from Dictionary<string,int> map, centered in each column

RightSideLegendPainter

Renders a compact color-coded legend on the right side. Shows colored square + series name per entry. Click toggles visibility. Hidden series crossed-out. Supports overflow scrolling. Colors from ChartDefaultSeriesColors palette.

Interaction & Trackball

ChartInputHelper supports:

  • Trackball -- Crosshair overlay with multi-series tooltip values. Configurable colors and width.
  • MouseWheelZoom -- Ctrl+Wheel zooms in/out (configurable step percent)
  • MouseDragPan -- Click-drag pans the viewport (configurable factor)
  • KeyboardNavigation -- Arrow keys for pan, +/- for zoom, configurable step percent
  • PointHover -- DataPointToolTip on hover with point details
  • PointSelection -- Click to select with visual highlight, configurable selection color/marker
  • LegendIsolate -- Click legend item to isolate single series. Click again to restore all.

18 dedicated events enable full integration: CustomDrawSeries, InteractiveAreaHit, PointHit, AxisHit, LegendHit, PointHoverChanged, ViewportChanged, TrackballDataCollected, SeriesIsolationChanged, SelectionChanged, ContextMenuRequested, KeyboardFocusChanged, PerformanceOptimized, NonColorStateCommunicationChanged, LabelOptimizationApplied, StreamingDataAppended, IncrementalRenderCompleted.

Code Examples

Basic Bar Chart

var chart = new BeepChart();
chart.ChartType = ChartType.Bar;
chart.ChartTitle = "Quarterly Revenue";
chart.ShowLegend = true;
chart.LegendPlacement = LegendPlacement.Right;

chart.DataSeries.Add(new ChartDataSeries
{
    SeriesName = "Q1 Revenue",
    DataPoints = new List<ChartDataPoint>
    {
        new ChartDataPoint { X = "Product A", Y = "1200" },
        new ChartDataPoint { X = "Product B", Y = "800" },
        new ChartDataPoint { X = "Product C", Y = "1500" }
    }
});

Large Dataset with Performance Mode

chart.EnablePerformanceMode = true;
chart.LargeDatasetThreshold = 50000;
chart.EnablePointCulling = true;
chart.EnableVertexSimplification = true;
chart.ResetViewport();

if (BeepChartPerformanceHelper.ShouldOptimizePerformance(
    chart.GetTotalPointCount(), chart.LargeDatasetThreshold))
{
    float cost = BeepChartPerformanceHelper.EstimateRenderingCost(
        chart.GetTotalPointCount(), chart.Width);
    Console.WriteLine($"Rendering cost: {cost:P0}");
}

Real-Time Streaming

chart.EnableRealTimeStreaming = true;
chart.StreamRenderThrottleMs = 50;   // render batch interval
chart.MaxStreamPointsPerSeries = 1000; // rolling window

chart.StreamingDataAppended += (s, e) =>
    Console.WriteLine($"Stream: {e.NewPointCount} points");

chart.AppendDataPoint(0, new ChartDataPoint
    { X = DateTime.Now.ToString("o"), Y = sensorValue.ToString() });

chart.EnableMouseWheelZoom = true;
