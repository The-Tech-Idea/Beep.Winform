Overview

The BeepMarquee control provides an animated ticker/scrolling display with 8 distinct item renderer styles. Each renderer implements IMarqueeItemRenderer and is resolved via MarqueePainterFactory. The factory creates renderer instances on demand, falling back to DefaultMarqueePainter if no match is found.

Renderers receive a MarqueeRenderContext that captures the current scroll position, theme colors, animation offset, and direction. This enables styles ranging from simple text scrolling to stock tickers with green/red arrows and news banners with separator dots.

IMarqueeItemRenderer Contract

public interface IMarqueeItemRenderer
{
    string Name { get; }
    Size Measure(Graphics g, MarqueeItem item, MarqueeRenderContext context);
    void Draw(Graphics g, MarqueeItem item, Rectangle bounds, MarqueeRenderContext context);
}
MemberTypeDescription
NamestringUnique renderer name used for lookup (e.g., "Default", "StockTicker", "NewsBanner")
MeasureMethodComputes the pixel size of a marquee item given the current context (font, spacing, style)
DrawMethodRenders the item within its bounds: text, icons, backgrounds, arrows, separators, badges

MarqueePainterFactory

MarqueePainterFactory resolves IMarqueeItemRenderer instances by name. It maintains a static dictionary of all 8 renderers and provides a Create method with fallback logic.

MemberSignatureDescription
Createstatic IMarqueeItemRenderer Create(string name)Returns the renderer instance for the given name; returns DefaultMarqueePainter if name is invalid
Registerstatic void Register(string name, IMarqueeItemRenderer renderer)Registers a custom renderer, overriding the default for that name

MarqueeRenderContext

MarqueeRenderContext is passed to every Measure and Draw call and captures the current state of the marquee animation.

PropertyTypeDescription
ThemeThemeCurrent theme object for color resolution
FontFontBase font for item text rendering
TextColorColorDefault text color from theme
AccentColorColorAccent/highlight color for emphasis (arrows, badges, backgrounds)
ScrollPositionfloatCurrent scroll offset in pixels; used to calculate item positions for smooth animation
IsVerticalboolTrue for vertical scrolling (Vertical renderer); false for horizontal
GapintPixel gap between consecutive items
AnimationOffsetfloatAdditional animation phase offset (in radians) for effects like pulsing or wave motion

Examples

Style Switching with Items

var marquee = new BeepMarquee();

marquee.Items.Add(new MarqueeItem { Text = "AAPL 192.53 ▲ 2.1%", Tag = "stock" });
marquee.Items.Add(new MarqueeItem { Text = "GOOGL 175.12 ▼ 0.8%", Tag = "stock" });
marquee.Items.Add(new MarqueeItem { Text = "MSFT 378.91 ▲ 1.4%", Tag = "stock" });
marquee.Items.Add(new MarqueeItem { Text = "TSLA 245.67 ▼ 3.2%", Tag = "stock" });

marquee.RendererName = "StockTicker";
marquee.ScrollSpeed = 60;
marquee.Gap = 32;

marquee.Items.Clear();
marquee.Items.Add(new MarqueeItem { Text = "BREAKING: Market opens higher on tech rally", Tag = "news" });
marquee.Items.Add(new MarqueeItem { Text = "SPORTS: Local team advances to finals", Tag = "news" });
marquee.Items.Add(new MarqueeItem { Text = "WEATHER: Sunny, high of 75°F", Tag = "news" });
marquee.RendererName = "NewsBanner";

marquee.RendererName = "Card";
marquee.Items.Clear();
for (int i = 0; i < 10; i++)
    marquee.Items.Add(new MarqueeItem { Text = $"User {i} completed task #{i * 7}" });
marquee.ScrollSpeed = 40;

marquee.RendererName = "Vertical";
marquee.IsVertical = true;
marquee.Height = 120;
