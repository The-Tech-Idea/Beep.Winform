Theme Types

Six concrete theme implementations — Ubuntu (26 files), GNOME (30 files), Cyberpunk (30 files), Candy, Zen, CyberpunkNeon — plus BeepThemesManager singleton

Overview

Six concrete theme implementations are available, each extending BeepTheme and overriding approximately 200 token properties. Themes provide cohesive visual language: button colors, fonts, border radii, surface colors, chart palettes, and more. Each theme is composed of multiple partial class files, totaling over 120 files across all themes. The BeepThemesManager singleton manages registration, lookup, and active theme switching.

BeepThemesManager

The BeepThemesManager is a singleton that manages all theme registration, lookup, and active theme switching. Controls query it for the current active theme during painting. When the theme changes, all controls subscribing to the ThemeChanged event invalidate and repaint themselves.

public sealed class BeepThemesManager
{
    private static readonly Lazy<BeepThemesManager> _instance =
        new Lazy<BeepThemesManager>(() => new BeepThemesManager());

    public static BeepThemesManager Instance => _instance.Value;

    public IBeepTheme CurrentTheme { get; private set; }
    public string CurrentThemeName => CurrentTheme?.ThemeName ?? "Default";

    public event EventHandler ThemeChanged;

    private readonly Dictionary<string, IBeepTheme> _themes = new(StringComparer.OrdinalIgnoreCase);

    public void RegisterTheme(IBeepTheme theme)
    {
        if (theme == null) throw new ArgumentNullException(nameof(theme));
        _themes[theme.ThemeName] = theme;
    }

    public bool SetTheme(string themeName)
    {
        if (!_themes.TryGetValue(themeName, out var theme))
            return false;

        var previous = CurrentTheme;
        CurrentTheme = theme;

        if (previous != CurrentTheme)
            ThemeChanged?.Invoke(this, EventArgs.Empty);

        return true;
    }

    public IBeepTheme GetTheme(string themeName)
    {
        return _themes.TryGetValue(themeName, out var theme) ? theme : null;
    }

    public IEnumerable<string> GetThemeNames() => _themes.Keys;

    public void InitializeDefaultThemes()
    {
        RegisterTheme(new UbuntuTheme());
        RegisterTheme(new GNOMETheme());
        RegisterTheme(new CyberpunkTheme());
        RegisterTheme(new CandyTheme());
        RegisterTheme(new ZenTheme());
        RegisterTheme(new CyberpunkNeonTheme());
        SetTheme("Ubuntu"); // default
    }
}
MemberDescription
InstanceThread-safe singleton accessor (Lazy<T>)
CurrentThemeThe currently active IBeepTheme. All controls read from this.
CurrentThemeNameConvenience property returning CurrentTheme.ThemeName or "Default"
ThemeChangedEvent raised when the active theme changes. Controls handle by invalidating/repainting.
RegisterTheme(IBeepTheme)Adds a theme to the registry by its ThemeName. Overwrites if name exists.
SetTheme(string)Activates a theme by name. Returns true if successful. Fires ThemeChanged.
GetTheme(string)Retrieves a registered theme by name. Returns null if not found.
GetThemeNames()Enumerates all registered theme names for UI dropdown/list display.
InitializeDefaultThemes()Registers all 6 built-in themes and sets Ubuntu as default. Called at app startup.

Examples

// Global theme switching
BeepThemesManager.Instance.InitializeDefaultThemes(); // register all 6 themes
BeepThemesManager.Instance.SetTheme("Cyberpunk");

// Check current theme
string currentTheme = BeepThemesManager.Instance.CurrentThemeName;
Console.WriteLine($"Active theme: {currentTheme}"); // "Cyberpunk"

// Register a custom theme
var customTheme = new BeepTheme
{
    ThemeName = "CorporateBlue",
    ButtonBackColor = Color.FromArgb(0, 82, 204),
    ButtonForeColor = Color.White,
    ButtonBorderColor = Color.FromArgb(0, 70, 180),
    ButtonHoverColor = Color.FromArgb(0, 100, 230),
    BackgroundColor = Color.FromArgb(244, 245, 247),
    ForegroundColor = Color.FromArgb(23, 43, 77),
    BorderRadius = 4,
    DefaultFont = new Font("Segoe UI", 9f),
    TitleFont = new Font("Segoe UI", 12f, FontStyle.SemiBold),
    // ... override all needed tokens
};
BeepThemesManager.Instance.RegisterTheme(customTheme);
BeepThemesManager.Instance.SetTheme("CorporateBlue");

// Subscribe to theme changes
BeepThemesManager.Instance.ThemeChanged += (s, e) =>
{
    Console.WriteLine($"Theme switched to: {BeepThemesManager.Instance.CurrentThemeName}");
    // Re-apply any custom overrides that got wiped
};

// Theme selection UI
var themeNames = BeepThemesManager.Instance.GetThemeNames().ToList(); // [Ubuntu, GNOME, Cyberpunk, Candy, Zen, CyberpunkNeon]
comboBoxTheme.Items.AddRange(themeNames);
comboBoxTheme.SelectedValueChanged += (s, e) =>
{
    BeepThemesManager.Instance.SetTheme(comboBoxTheme.SelectedItem.ToString());
};

// Per-control theme inspection
var theme = BeepThemesManager.Instance.CurrentTheme;
Console.WriteLine($"Button color: {theme.ButtonBackColor}");
Console.WriteLine($"Grid header: {theme.GridHeaderBackColor}");
Console.WriteLine($"Chart palette[0]: {theme.ChartPalette[0]}");
