Overview

BeepBottomBar (class name BottomBar) is a feature-rich bottom navigation control built on BaseControl. It displays a horizontal row of navigation items as SimpleItem objects with icons, text labels, and optional badges. The control supports 10 distinct visual styles driven by a painter architecture, smooth animated selection indicators, centered floating call-to-action (CTA) items, and accessibility-compliant keyboard navigation.

Each visual style is implemented as a dedicated painter class, enabling drastically different appearances while sharing the same item model and event pipeline. The control auto-docks to the bottom of its parent container.

Inheritance: Component → Control → BaseControl → BottomBar  |  Designer: ControlDesigner (Visual Studio toolbox support)

Key Features

  • 10 visual styles — Classic, FloatingCTA, Bubble, Pill, Diamond, NotionMinimal, MovableNotch, OutlineFloatingCTA, SegmentedTrack, and GlassAcrylic
  • Smooth selection animation — Eased quartic interpolation animates the indicator between items with configurable AnimationDuration
  • Centered floating CTA — Designate any item as a CTA with CTAIndex; it renders raised above the bar with shadow effects
  • Popup child menus — Items with Children collections trigger popup sub-menus on press-and-hold or long-click
  • Responsive layout engine — BeepBottomBarLayoutHelper dynamically spaces items with configurable width factors for selected and CTA items
  • Label visibility policy — Choose Always, SelectedOnly, or IconOnly label display via LabelPolicy
  • Theme-aware rendering — Navigation-specific theme tokens drive bar, hover, badge, and shadow colors
  • Keyboard navigation — Full arrow key, Home/End, Space/Enter, and Escape support for accessible operation
  • Accessibility — Custom AccessibleObject hierarchy with MenuBar and MenuItem roles, live region descriptions
  • Continuous animation — Built-in ticker timer drives pulse effects for hover and active states
  • Tooltip integration — Hover tooltips show item text, sub-text, badge info, and child item counts
  • HiDPI scaling — DPI-aware minimum bar height scaling with DpiScalingHelper

Visual Styles

The BottomBarStyle enum offers 10 distinct visual styles. Each is rendered by a dedicated painter class:

StyleValueDescription
Classic0Standard flat bottom bar with pill-shaped selection indicator. Default style.
FloatingCTA1Raises the CTA item above the bar with a floating pill shape and drop shadow.
Bubble2Selection indicator appears as a colored bubble that stretches behind the active item.
Pill3Rounded pill-shaped bar with the active indicator sliding within a capsule.
Diamond4Angled cut shape for the selection indicator with a diamond-like appearance.
NotionMinimal5Minimalist style inspired by Notion's sidebar: thin accent underline on selected item.
MovableNotch6Animated notch cutout that moves along the top edge to highlight the selected item.
OutlineFloatingCTA7Center CTA rendered with an animated glow halo, inner highlight, and ring stroke outline.
SegmentedTrack8Horizontal segmented track bar with a sliding capsule indicator. iOS style.
GlassAcrylic9Translucent glass/acrylic effect with blur and configurable opacity.

Theming

BeepBottomBar deeply integrates with the Beep theme system using navigation-specific tokens when available, falling back to surface and accent colors:

  • NavigationBackColor / NavigationForeColor — Bar background and text colors
  • NavigationHoverBackColor / NavigationHoverForeColor — Hover state colors for items
  • BorderColor / ActiveBorderColor — Navigation border color derivation
  • BadgeBackColor / BadgeForeColor — Item badge colors
  • AccentColor — Drives the selection indicator, CTA highlight, and icon tinting
  • OnPrimaryColor — Icon/text color on the accent background
BeepThemesManager.ApplyTheme("AzureDark"); // Bottom bar and all painters update
bottomBar.ApplyTheme(); // Force per-instance theme refresh

Examples

Basic Bottom Navigation

var bottomBar = new BottomBar
{
    BarHeight = 64,
    BarStyle = BottomBarStyle.Classic,
    Dock = DockStyle.Bottom
};

bottomBar.Items.Add(new SimpleItem { Text = "Home", ImagePath = Svgs.Home });
bottomBar.Items.Add(new SimpleItem { Text = "Search", ImagePath = Svgs.Search });
bottomBar.Items.Add(new SimpleItem { Text = "Profile", ImagePath = Svgs.User });

bottomBar.ItemClicked += item => Console.WriteLine($"Navigated to: {item.Text}");

Floating CTA with Custom Styling

var bottomBar = new BottomBar
{
    BarStyle = BottomBarStyle.FloatingCTA,
    CTAIndex = 2,              // Center item as CTA
    CTAWidthFactor = 1.8f,
    ShowCTAShadow = true,
    CTAShadowYOffset = 10,
    AccentColor = Color.FromArgb(255, 65, 108)
};

bottomBar.Items.Add(new SimpleItem { Text = "Feed", ImagePath = Svgs.Home });
bottomBar.Items.Add(new SimpleItem { Text = "Explore", ImagePath = Svgs.Search });
bottomBar.Items.Add(new SimpleItem { Text = "Add", ImagePath = Svgs.Plus });
bottomBar.Items.Add(new SimpleItem { Text = "Messages", ImagePath = Svgs.Chat });
bottomBar.Items.Add(new SimpleItem { Text = "Account", ImagePath = Svgs.User });

Popup Child Menus

var moreItem = new SimpleItem
{
    Text = "More",
    ImagePath = Svgs.More,
    Children = new List<SimpleItem>
    {
        new SimpleItem { Text = "Settings", ImagePath = Svgs.Settings },
        new SimpleItem { Text = "Help", ImagePath = Svgs.Help },
        new SimpleItem { Text = "About", ImagePath = Svgs.Info }
    }
};
bottomBar.Items.Add(moreItem);

bottomBar.PopupRequested += (s, e) => Console.WriteLine($"Popup for: {e.Item.Text}");
bottomBar.PopupClosed += (s, e) => Console.WriteLine("Popup dismissed");

iOS-Style Segmented Track

var iosBar = new BottomBar
{
    BarStyle = BottomBarStyle.SegmentedTrack,
    SegmentedTrackHeight = 8,
    SegmentedIndicatorWidth = 48,
    SelectedWidthFactor = 1.1f,
    AnimationDuration = 300
};

iosBar.Items.Add(new SimpleItem { Text = "Today", ImagePath = Svgs.Calendar });
iosBar.Items.Add(new SimpleItem { Text = "Games", ImagePath = Svgs.Gamepad });
iosBar.Items.Add(new SimpleItem { Text = "Apps", ImagePath = Svgs.Grid });
iosBar.Items.Add(new SimpleItem { Text = "Search", ImagePath = Svgs.Search });
