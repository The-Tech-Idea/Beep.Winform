BeepChevronButton

Directional chevron button with checkable toggle state, arrow cycling, ripple effect, and 3 visual variants

Overview

BeepChevronButton draws an arrow/chevron button with configurable direction and size. Supports a checkable toggle state, arrow direction cycling on click, ripple animation, and three visual variants (Default, Subtle, Outlined).

Namespace: TheTechIdea.Beep.Winform.Controls
Inheritance: BaseControl → BeepChevronButton

Examples

Expand/Collapse Toggle

var chevron = new BeepChevronButton { Direction = ChevronDirection.Right, IsCheckable = true };
chevron.Click += (s, e) => {
    panel.Visible = chevron.IsChecked;
    chevron.Direction = chevron.IsChecked ? ChevronDirection.Down : ChevronDirection.Right;
};
