BeepChipListBox

A combined control integrating BeepListBox and BeepMultiChipGroup with optional search, bidirectional selection synchronization, and style coordination

Overview

BeepChipListBox is a composite control that combines a BeepListBox, a BeepMultiChipGroup, and a BeepTextBox search bar into a single unified component. It inherits from BaseControl and resides in TheTechIdea.Beep.Winform.Controls.CombinedControls. The control provides bidirectional selection synchronization via a SelectionSyncHelper, style coordination through ChipListBoxStyleCoordinator, and flexible layout arrangements (chips above list, chips below list, chips only, list only). The class is split across multiple partial files: Core, Properties, Events, Methods, and Drawing.

Key Features

  • Bidirectional selection sync between list box and chip group
  • Built-in search with auto-filtering of list items
  • Four layout modes: ChipsAboveList, ChipsBelowList, ChipsOnly, ListOnly
  • Style coordination system with presets (Default, Modern, Minimal, etc.)
  • Multi-select support with checkbox integration
  • Keyboard shortcuts: Ctrl+F (focus search), Escape (clear search), Ctrl+A (select all)
  • Divider with subtle shadow effects between sections
  • Full theme propagation to all child controls
  • Configurable chip style (Default, Rounded, Pill, etc.) and size (Small, Medium, Large)
  • Configurable list box type (Standard, Compact, Card, etc.)

Theming

The control fully propagates theme to all child controls. ApplyTheme() sets the divider color from _currentTheme.BorderColor, panel backgrounds from _currentTheme.PanelBackColor, and forwards the theme name to _searchBox, _searchIcon, _chipGroup, and _listBox. Border color changes on focus (uses PrimaryColor) for visual state indication.

Examples

// Create and configure a chip list box
var chipListBox = new BeepChipListBox
{
    Size = new Size(300, 400),
    StylePreset = ChipListBoxStylePreset.Modern,
    Layout = ChipListBoxLayout.ChipsAboveList,
    AllowMultiSelect = true,
    ShowCheckBoxes = true,
    ChipGroupTitle = "Selected Items",
    SearchPlaceholderText = "Filter items..."
};

// Add items
chipListBox.AddItems(new[]
{
    new SimpleItem { Text = "Design", Value = "design", ImagePath = Svgs.Paint },
    new SimpleItem { Text = "Development", Value = "dev", ImagePath = Svgs.Code },
    new SimpleItem { Text = "Testing", Value = "qa", ImagePath = Svgs.Bug },
    new SimpleItem { Text = "Documentation", Value = "docs", ImagePath = Svgs.File }
});

// Handle selection
chipListBox.SelectionChanged += (s, e) =>
{
    Console.WriteLine($"Selected {e.SelectedItems.Count} items via {e.Source}");
};

// Handle chip removal
chipListBox.ChipRemoved += (s, item) =>
{
    Console.WriteLine($"Chip removed: {item.Text}");
};

// Handle search
chipListBox.SearchTextChanged += (s, e) =>
{
    Console.WriteLine($"Searching for: {e.SearchText}");
};

// Keyboard: Ctrl+F focuses search, Escape clears, Ctrl+A selects all
this.Controls.Add(chipListBox);

Architecture

BeepChipListBox is a composite control composed of four panel regions stacked vertically: search panel (top, optional), chip panel (with BeepMultiChipGroup), divider panel, and list panel (with BeepListBox, dock fill). A SelectionSyncHelper manages bidirectional state synchronization between the list box and chip group. The ChipListBoxStyleCoordinator static class provides style mappings between ListBoxType and ChipStyle values. Layout reordering is achieved by clearing and re-adding child controls to maintain correct Z-order based on ChipListBoxLayout enum. The control is implemented across five partial class files for maintainability: Core (fields, constructor, initialization, layout, theme), Properties (all public properties), Events (event definitions and keyboard/focus handling), Methods (selection, item management, search, sync, style, data binding) and Drawing (paint overrides and visual state).
