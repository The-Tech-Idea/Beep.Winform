BeepCircularButton

Fully circular button with image support, configurable text positioning, and color-set mode

TOC

  • Overview
  • Properties
  • Examples

Overview

BeepCircularButton renders a fully circular button with optional image and text. Supports text positioning relative to the circle (Top, Bottom, Left, Right, None), a color-set mode for palette pickers, and full theme integration.

Namespace: TheTechIdea.Beep.Winform.Controls
Inheritance: BaseControl → BeepCircularButton

Properties

PropertyTypeDefaultDescription
ImagePathstring""Path to the image displayed inside the circle.

TextAlignContentAlignmentCenterAlignment of text relative to its position.
HideTextboolfalseHides the text, showing only the circle+image.
ShowBorderbooltrueWhether to draw a border around the circle.
ApplyThemeOnImagebooltrueApplies theme colors to the image.
IsForColorSetboolfalseColor picker mode; fills circle with BackColor.

Examples

Photo Avatar Button

var avatar = new BeepCircularButton {
    ImagePath = "profile.jpg", TextLocation = TextLocation.Bottom,
    Text = "John Doe", ShowBorder = false };
avatar.Click += (s, e) => OpenProfile();

Color Picker

var colorBtn = new BeepCircularButton {
    IsForColorSet = true, BackColor = Color.Red, ShowBorder = true };
colorBtn.Click += (s, e) => PickColor(colorBtn.BackColor);
