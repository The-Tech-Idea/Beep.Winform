Overview

Namespace: TheTechIdea.Beep.Winform.Controls

Assembly: TheTechIdea.Beep.Winform.Controls

Inheritance: BaseControl

Source: 925 lines in a single file (Dates\BeepDatePickerView.cs)

The BeepDatePickerView is a modern, self-contained calendar view control optimized for date selection with a clean, responsive design. It renders a full month calendar with a month/year header, day-of-week headers, a 6-row date grid, a selected-date display bar, and footer action links (Clear/Done). An optional time spinner allows minute-level adjustments. All interaction is handled via hit-test rectangles — there are no child controls. The control is designed for embedding in popups, dropdowns, BeepGridPro cells, and containers, offering both a DrawContent override (for BaseControl) and a public Draw method (for container hosting).

It fires explicit OkClicked and CancelClicked events (with DateTimeDialogResultEventArgs carrying the selected datetime, context, and confirmation flag) while maintaining backward-compatible DateTimeSelected and Cancelled events.

Key Features

  • Self-contained rendering with zero child controls — all UI drawn and hit-tested via rectangles
  • Month/year header with left/right chevron navigation and hover/press visual feedback
  • Monday-start 7-column calendar grid with 6 rows, showing leading/trailing month days in dimmed colors
  • Selected date highlight with rounded filled background and contrast-aware text color
  • Today indicator with a bordered ring
  • Selected date display bar with blue rounded background and optional time spinner
  • Time spinner: up/down chevrons with 1-minute increments, hover and press states
  • "Clear" and "Done" footer links with hover underline and background effects
  • Explicit OK/Cancel API via OkClicked / CancelClicked events with context payload
  • Backward-compatible DateTimeSelected / Cancelled events
  • Enforced minimum size (280x420) via SetBoundsCore and OnResize
  • Responsive layout that scales cell sizes within MinCell (28px) and BaseCellSize (36px) constraints
  • Theme integration via calendar-specific tokens with font auto-application
  • Smoothing and ClearType text rendering for crisp visuals
  • Double-buffered painting for flicker-free rendering
  • Compatible with BeepGridPro cell hosting via public Draw(Graphics, Rectangle) overrides

Layout Architecture

The control's visual layout is computed by UpdateModernLayout() and consists of distinct rectangular regions. No child controls are used — all hit detection is performed against these internal rectangles registered via the BaseControl AddHitArea helper:

RegionInternal FieldHit Action
Month/Year Header_headerRectContains PrevMonth/NextMonth chevron sub-rects
Day-of-Week Headers_daysHeaderRectNon-interactive: "Mo Tu We Th Fr Sa Su"
Date Grid (6 rows x 7 cols)_datesGridRectDate cell selection via HandleDateSelection
Selected Date Display_selectedDateDisplayRectContains TimeUp/TimeDown spinner sub-rects
Footer Button Bar_buttonsRectContains Clear (left) and Done (right) link rects
Prev Month Chevron_prevMonthRectNavigates display month -1
Next Month Chevron_nextMonthRectNavigates display month +1
Clear Link_clearRectSets SelectedDateTime to null, fires CancelClicked
Done Link_doneRectFires OkClicked
Time Spinner Up_timeUpRectAdds 1 minute to selected time
Time Spinner Down_timeDownRectSubtracts 1 minute from selected time

The layout is fully responsive: cell width is computed as Math.Max(MinCell, Math.Min(BaseCellSize, availableWidth / 7)) with the grid width snapped to cellWidth * 7. The control enforces a computed minimum height via CalculateMinimumRequiredHeight() that accounts for all regions plus padding.

All 5 fonts are only applied when UseThemeFont is true (default). The ApplyTheme() method then calls UpdateModernLayout() to recompute geometry and InvalidateOnce() for batched repaint. Contrast fallback is computed via ColorUtils.GetContrastColor() when a foreground color token is empty.

Examples

Basic Dropdown Hosting

var dateView = new BeepDatePickerView
{
    Context = "Due Date",
    ShowTime = true,
    SelectedDateTime = DateTime.Now
};
dateView.OkClicked += (s, e) =>
{
    Console.WriteLine($"Confirmed: {e.SelectedDateTime}, Context: {e.Context}");
};
dateView.CancelClicked += (s, e) =>
{
    Console.WriteLine($"Cancelled. Context: {e.Context}");
};

var form = new Form();
dateView.Dock = DockStyle.Fill;
form.Controls.Add(dateView);
form.ShowDialog();

Embedding in a ToolStripDropDown (Popup)

var dropDown = new ToolStripDropDown();
var dateView = new BeepDatePickerView
{
    Context = "Start Date",
    ShowTime = false,
    AutoSize = false,
    Size = new Size(320, 380)
};
dropDown.Items.Add(new ToolStripControlHost(dateView));
dateView.OkClicked += (s, e) => dropDown.Close();
dateView.CancelClicked += (s, e) => dropDown.Close();
dropDown.Show(PointToScreen(new Point(btn.Left, btn.Bottom)));

Setting Initial Date and Time

var dateView = new BeepDatePickerView
{
    Context = "Appointment",
    ShowTime = true,
    SelectedDateTime = new DateTime(2025, 12, 25, 14, 30, 0)
};
dateView.OkClicked += (s, e) =>
{
    MessageBox.Show($"Scheduled: {e.SelectedDateTime:g}");
};
this.Controls.Add(dateView);

Architecture

FileResponsibility
BeepDatePickerView.csSingle-file implementation (925 lines) containing all logic: fields, properties, events, constructor, layout computation, painting (header/days/dates/selected-date/footer/time-spinner), mouse handling (OnMouseDown/Up/Move/Leave), size enforcement, theming, and HSL color utilities

The control follows a hit-test-driven pattern where all mouse interactions are routed through BaseControl's AddHitArea / ClearHitList helpers. The _hitTest object tracks hovered and pressed states for visual feedback. Fallback direct rectangle containment checks in OnMouseDown guarantee action execution even if the hit helper misses.

The painting pipeline supports dual rendering contexts: the DrawContent override paints within BaseControl's DrawingRect, while the public Draw(Graphics, Rectangle) override supports BeepGridPro cell rendering with an explicit bounds rectangle.
