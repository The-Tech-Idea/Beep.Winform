Overview

BeepDateRangePicker is a composite control built on BaseControl that provides a compact start-to-end date range selector. It features two BeepTextBox inputs for manual date entry (yyyy-MM-dd format), two calendar trigger buttons that open a MonthCalendar popup, and a "to" label separator. The control exposes StartDate and EndDate as DateTime? properties, a read-only IsValidRange check, and a DateRangeChanged event.

Designed primarily for use in BeepGridPro filtering scenarios (e.g., a Between date filter), BeepDateRangePicker is a lightweight, self-contained control that supports preset ranges (Today, Last7Days, ThisMonth, etc.) via the SetPresetRange method and the DateRangePreset enum. The calendar popup positions itself beneath the appropriate calendar button and closes on date selection or lost focus.

Key Features

Dual Date Inputs

Two BeepTextBox fields with placeholder text ("Start date", "End date") accept dates in yyyy-MM-dd format. Dates are parsed with DateTime.TryParse for flexibility.

Calendar Popup

Dedicated calendar buttons (📅) open a MonthCalendar in a borderless popup form, positioned below the triggering button. Selecting a date auto-fills the corresponding text box.

8 Preset Ranges

The DateRangePreset enum provides quick-set ranges: Today, Yesterday, Last7Days, Last30Days, ThisMonth, LastMonth, ThisYear, LastYear.

Range Validation

The read-only IsValidRange property verifies both dates are set and StartDate <= EndDate.

Real-Time Events

DateRangeChanged event fires whenever either text box value changes or a calendar selection is made.

Theming

Inherits BaseControl theming. All child BeepControls (BeepTextBox, BeepButton, BeepLabel) are marked IsChild = true for theme propagation.

Properties

PropertyTypeAccessDefaultDescription
StartDate DateTime? Get / Set null The start date of the range. On get, parses _startDateTextBox.Text using DateTime.TryParse — returns null if the text is not a valid date. On set, writes the date as yyyy-MM-dd format (empty string for null).
EndDate DateTime? Get / Set null The end date of the range. Same parse/set semantics as StartDate using _endDateTextBox.
IsValidRange bool Read false Returns true only if both start and end dates have values and StartDate <= EndDate. Returns false if either date is null or if start is after end.
Inherited Properties: As a BaseControl, this control inherits Theme, CurrentTheme, UseThemeColors, IsFrameless, ShowAllBorders, ShowShadow, and all standard UserControl properties.

Events

EventArgs TypeDescription
DateRangeChanged EventArgs Fires when either the start or end date text changes (TextChanged on either BeepTextBox) or when a date is selected from the calendar popup (which programmatically sets the text box value). Use this to reactively enable/disable date-range-dependent UI elements.

Methods

Public Methods

MethodSignatureDescription
Clear public void Clear() Sets both _startDateTextBox.Text and _endDateTextBox.Text to string.Empty. Does not explicitly fire DateRangeChanged but the TextChanged handlers will trigger.
SetPresetRange public void SetPresetRange(DateRangePreset preset) Calculates and sets StartDate and EndDate based on the selected DateRangePreset value. The end date is always DateTime.Today (except yesterday/last month/last year where it is adjusted). See DateRangePreset Enum for all presets.

Private Methods

MethodDescription
InitializeControls()Creates all five child controls: _startDateTextBox (120px, placeholder "Start date"), _startCalendarButton (30×30, text "📅"), _toLabel (30×30, "to", center-aligned), _endDateTextBox (120px, placeholder "End date"), _endCalendarButton (30×30). All are marked IsChild = true and added to Controls.
LayoutControls()Positions controls horizontally with 5px margins: _startDateTextBox → _startCalendarButton → _toLabel → _endDateTextBox → _endCalendarButton. Sets the control's Height and Width based on the accumulated positions.

Layout & Internal Structure

The control is built in the default constructor (BeepDateRangePicker()), which sets IsFrameless = true, ShowAllBorders = false, ShowShadow = false, then calls InitializeControls(), LayoutControls(), and HookEvents().

Five child controls are created and positioned in a single horizontal row:

The calendar popup is a standalone Form (not a child control) created on demand and destroyed when closed. It uses FormBorderStyle.None, TopMost = true, and is positioned at the screen coordinates below the clicked 📅 button using button.PointToScreen().

Examples

Example 1: Basic Usage

Create and Use the Date Range Picker
var dateRange = new BeepDateRangePicker
{
    Location = new Point(12, 12)
};

// React to changes
dateRange.DateRangeChanged += (s, e) =>
{
    if (dateRange.IsValidRange)
    {
        Console.WriteLine(
            $"Valid range: {dateRange.StartDate:yyyy-MM-dd}"
            + $" → {dateRange.EndDate:yyyy-MM-dd}");
        LoadDataForRange(dateRange.StartDate.Value,
                         dateRange.EndDate.Value);
    }
};

this.Controls.Add(dateRange);

Example 2: Preset Buttons

Quick Filter Presets
var dateRange = new BeepDateRangePicker();

// Button: "Last 7 Days"
btnLast7Days.Click += (s, e) =>
    dateRange.SetPresetRange(DateRangePreset.Last7Days);

// Button: "This Month"
btnThisMonth.Click += (s, e) =>
    dateRange.SetPresetRange(DateRangePreset.ThisMonth);

// Check validity before applying
btnApply.Click += (s, e) =>
{
    if (dateRange.IsValidRange)
        ApplyFilter(dateRange.StartDate, dateRange.EndDate);
    else
        MessageBox.Show("Please set a valid date range.");
};

Example 3: Integration with BeepGridPro Filter Row

Using in a Grid Filter Row
// In a BeepFilterRow, when the operator is "Between"
// for a DateTime column:
var datePicker = new BeepDateRangePicker
{
    IsChild = true,
    IsFrameless = true
};

// When both dates are valid, construct the filter criteria
datePicker.DateRangeChanged += (s, e) =>
{
    if (datePicker.IsValidRange)
    {
        var criteria = new FilterCriteria(
            columnName: "OrderDate",
            op: FilterOperator.Between,
            value: datePicker.StartDate,
            value2: datePicker.EndDate
        );
        OnFilterCriteriaChanged(criteria);
    }
};

Architecture

BeepDateRangePicker — the composite date range control

BaseControl (Base) Base class providing theming infrastructure, border styling, and shadow support.

BeepButton (Buttons) Themed button used for calendar trigger icons.
BeepLabel Themed label used for the "to" separator.
