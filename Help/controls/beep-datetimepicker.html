Overview

Namespace: TheTechIdea.Beep.Winform.Controls.Dates

Assembly: TheTechIdea.Beep.Winform.Controls

Inheritance: BaseControl

The BeepDateTimePicker is a comprehensive, themeable date/time picker with no dropdown - all rendering is performed directly on the control surface using a painter/hit-handler architecture. It supports 18 distinct functional modes, each with dedicated painting and interaction logic. The control extends BaseControl and integrates with the Beep theming system for consistent visual styling.

The control is split across 7 partial files: Core (fields, events, constructor), Properties, Events (input handling), Methods (public API), Drawing (paint pipeline), Layout, and DateHandling. All calendar rendering is delegated to IDateTimePickerPainter implementations, while mouse/keyboard interaction is handled by IDateTimePickerHitHandler implementations.

Key Features

  • 18 functional modes: Single, SingleWithTime, Range, RangeWithTime, Multiple, Appointment, Timeline, Quarterly, Compact, ModernCard, DualCalendar, WeekView, MonthView, YearView, SidebarEvent, FlexibleRange, FilteredRange, and Header
  • Direct-rendering design - no dropdown, all calendar UI painted on the control surface
  • Pluggable painter architecture via IDateTimePickerPainter with 18 mode-specific painters
  • Pluggable hit-handler architecture via IDateTimePickerHitHandler for mode-specific interaction
  • Full theme support using calendar-specific theme tokens
  • Keyboard navigation: arrow keys for day, page up/down for month, Home for today
  • Quick selection buttons: Today, Tomorrow, Yesterday, This Week, This Month
  • Time picker with configurable interval and min/max time constraints
  • Range selection with auto-swapping of start/end
  • Multiple date selection with toggle and limit support
  • Min/Max date and time constraints
  • DPI-aware font scaling (optional)
  • Custom date format and first day of week
  • Appointment mode with time slot scrolling
  • Flexible range mode with tabs and quick date options
  • Filtered range with holiday/date filter support
  • Hosted in ToolStripDropDown for popup scenarios

Theming

BeepDateTimePicker uses calendar-specific theme tokens from BeepTheme. The ApplyTheme() method reads these tokens:

Theme TokenUsage
CalendarBackColorControl background color
CalendarForeColorRegular (unselected) date text color
CalendarTitleForColorHeader text and navigation button color
CalendarDaysHeaderForColorDay name header color (Mon, Tue, etc.)
CalendarSelectedDateBackColorSelected date highlight background
CalendarSelectedDateForColorText color on selected date
CalendarTodayForeColorToday indicator ring/border color
CalendarBorderColorBorder color around calendar elements
CalendarHoverBackColorHover state background for date cells
DateFontFont for calendar date numbers
CalendarTitleFontFont for month/year header
DaysHeaderFontFont for day-of-week header

The font is set via BeepFontManager.ToFont() with a fallback chain: theme font → BeepFontManager default → Segoe UI 9.75pt. The UseThemeFont property controls whether theme fonts are applied.

Examples

Basic Single Date Selection

var picker = new BeepDateTimePicker();
picker.Mode = DatePickerMode.Single;
picker.SelectedDate = DateTime.Today;
picker.DateChanged += (s, e) => {
    Console.WriteLine($"Selected: {e.Date:d}");
};
this.Controls.Add(picker);

Date Range with Time

var rangePicker = new BeepDateTimePicker();
rangePicker.Mode = DatePickerMode.RangeWithTime;
rangePicker.ShowTime = true;
rangePicker.MinDate = DateTime.Today;
rangePicker.MaxDate = DateTime.Today.AddMonths(3);
rangePicker.RangeChanged += (s, e) => {
    Console.WriteLine($"Range: {e.Date:d} - {e.RangeEnd:d}");
    Console.WriteLine($"Times: {e.Time} - ...");
};
rangePicker.SetRange(DateTime.Today, DateTime.Today.AddDays(7));
this.Controls.Add(rangePicker);

Multiple Date Selection

var multiPicker = new BeepDateTimePicker();
multiPicker.Mode = DatePickerMode.Multiple;
multiPicker.MaxMultipleSelectionCount = 5;
multiPicker.SelectedDates = new List<DateTime> {
    DateTime.Today, DateTime.Today.AddDays(2), DateTime.Today.AddDays(5)
};
multiPicker.DateChanged += (s, e) => {
    Console.WriteLine($"Total selected: {multiPicker.SelectedDates.Count}");
};
this.Controls.Add(multiPicker);

Appointment Mode with Time Slots

var appointmentPicker = new BeepDateTimePicker();
appointmentPicker.Mode = DatePickerMode.Appointment;
appointmentPicker.TimeStartHour = 8;
appointmentPicker.TimeIntervalMinutes = 15;
appointmentPicker.SelectedDateTime = new DateTime(2025, 6, 15, 10, 30, 0);
appointmentPicker.DateChanged += (s, e) => Console.WriteLine($"Date: {e.Date:d}");
appointmentPicker.TimeChanged += (s, e) => Console.WriteLine($"Time: {e.Time}");
this.Controls.Add(appointmentPicker);
