Overview

BeepDock is a modern, customizable dock control inspired by macOS-style docks. It renders items using pluggable painters, supports spring-animation effects on hover, drag-drop reordering, badge notifications, progress indicators, tooltips, keyboard navigation, and accessibility features. The control inherits from BaseControl and integrates with the Beep theming system.

Key Features

Painter-Based Rendering

Pluggable painters via IDockPainter with factory support for multiple visual styles including AppleDock, Material, Fluent, and Classic

Smooth Spring Animations

60 FPS spring-physics animation timer with configurable speed and max scale for hover magnification effects

Drag-Drop Reordering

Built-in drag initiation with hysteresis, visual drag feedback, drop indicators, and complete item reordering support

Badge Notifications

Automatic badge rendering with color-coded notification types (Info, Success, Warning, Error) and count display

Progress Indicators

Determinate and indeterminate circular progress arcs painted around individual dock items

Keyboard Navigation

Full arrow-key navigation, Home/End, Enter/Space activation, number-key quick access (1-9), Tab/Shift+Tab, and Escape to clear focus

Overflow Handling

Automatic overflow affordance popup when items exceed available space, with ellipsis button and flyout menu

Accessibility

UIAccessibility role set to ToolBar, descriptive accessible names, high-contrast mode support, and screen-reader announcements

Theming

BeepDock integrates with the Beep theming system through its ApplyTheme() override. Theme colors are applied via DockThemeHelpers and fonts via DockFontHelpers. The control maintains a frameless appearance (IsFrameless = true, ShowAllBorders = false, IsBorderAffectedByTheme = false) unless explicitly configured otherwise.

Both StyleProfile (DockStyleConfig) and ColorProfile (DockColorConfig) are expandable designer properties that can override individual theme aspects for precise visual control. When no explicit background color is set, the dock falls back to the parent control's background color.

Applying a Style Profile

var dock = new BeepDock();
dock.StyleProfile = new DockStyleConfig
{
    DockStyle = DockStyle.Material,
    RecommendedItemSize = 48,
    RecommendedDockHeight = 64,
    RecommendedMaxScale = 1.3f,
    ShowShadow = false
};

Applying a Color Profile

dock.ColorProfile = new DockColorConfig
{
    BackgroundColor = Color.FromArgb(30, 30, 30),
    ForegroundColor = Color.White,
    ItemSelectedColor = Color.FromArgb(0, 122, 255),
    IndicatorColor = Color.Cyan
};

Examples

Basic Dock Setup

var dock = new BeepDock
{
    Dock = DockStyle.Bottom,
    Height = 72,
    DockStyleType = DockStyle.AppleDock,
    ItemSize = 56,
    ItemSpacing = 8,
    ShowTooltips = true,
    ShowBadges = true,
    EnableReorder = true,
    EnableOverflow = true
};

dock.Items.Add(new SimpleItem { Text = "Home", ImagePath = "home.svg" });
dock.Items.Add(new SimpleItem { Text = "Mail", ImagePath = "mail.svg" });
dock.Items.Add(new SimpleItem { Text = "Calendar", ImagePath = "calendar.svg" });
dock.Items.Add(new SimpleItem { Text = "Settings", ImagePath = "settings.svg" });

dock.ItemClicked += (s, e) =>
{
    Console.WriteLine($"Clicked: {e.Item?.Text}");
};

this.Controls.Add(dock);

Notifications and Badges

// Enable badges
dock.ShowBadges = true;

// Set a simple badge count
dock.SetBadgeCount(mailItem, 5);

// Set a full notification with type
dock.SetNotification(mailItem, "New Messages", "You have 3 unread messages", NotificationType.Info);
dock.SetNotification(errorItem, "Build Failed", "Check error log", NotificationType.Error);

// Handle notification events
dock.NotificationAdded += (s, e) =>
{
    Console.WriteLine($"Notification for {e.Item.Text}: {e.Notification.Title}");
};

// Clear when viewed
dock.ClearNotification(mailItem);

// Progress indicators
dock.SetProgress(downloadItem, 45.0f, "Downloading...");
dock.SetIndeterminateProgress(uploadItem, "Uploading...");
dock.ClearProgress(downloadItem);

Drag-Drop and Reordering

var dock = new BeepDock
{
    EnableDrag = true,       // Allow dragging items externally
    EnableReorder = true     // Allow reordering within the dock
};

dock.DragStarted += (s, e) =>
{
    Console.WriteLine($"Drag started: {e.Item?.Text}");
};

dock.ItemReordered += (s, e) =>
{
    Console.WriteLine($"{e.Item?.Text} moved from {e.OldIndex} to {e.NewIndex}");
};

// External file drop support
dock.FilesDropped += (s, e) =>
{
    foreach (var file in e.Files)
    {
        var item = new SimpleItem
        {
            Text = Path.GetFileName(file),
            ImagePath = file,
            GuidId = Guid.NewGuid().ToString()
        };
        dock.AddItem(item);
    }
};

Keyboard Navigation

// Keyboard navigation is enabled by default (TabStop = true)
// Supported keys:
//   Left/Up    — Navigate to previous item
//   Right/Down — Navigate to next item
//   Home       — Jump to first item
//   End        — Jump to last item
//   Enter/Space— Activate (click) focused item
//   Escape     — Clear focus
//   Tab/Shift+Tab — Navigate next/previous
//   D1-D9      — Quick-select items 1-9
//   Apps/F10+Shift — Context menu for focused item

// Programmatic focus control
dock.FocusedIndex = 2;          // Focus third item
dock.KeyboardNavigationEnabled = false; // Disable keyboard nav

// Override context menu
// BeepDock provides a default context menu via the virtual ShowContextMenu method
