Overview

BeepDockingManager is a non-visual component (IComponent) that orchestrates a Krypton-style docking system. Drag it from the toolbox onto a form, then add DockPanel components to build complex MDI/docking layouts.

Namespace: TheTechIdea.Beep.Winform.Controls.Docking
Assembly: TheTechIdea.Beep.Winform.Controls.dll
Inheritance: IComponent → BeepDockingManager
Implements: IDisposable, IDockDragHost

Organized across 3 partial files plus supporting runtime, painter, layout, and model subsystems in the Docking namespace.

Key Features

  • Four panel states — Docked, Tabbed, Floating, Auto-Hidden (Visual Studio parity)
  • Component tray integration — Drop from toolbox onto form; manages panels as design-time components
  • Krypton-compatible event model — Cancelable request events (PageCloseRequest, PageFloatingRequest, etc.)
  • Drag-to-float/dock — Panel drag creates floating window; drag to edges or tab strip to re-dock
  • Auto-hide strips — Collapsed panels along each edge with slide-out peek behavior
  • Splitter management — Resizable splitter bars between docked groups
  • Layout persistence — Save/restore layout trees via DockLayoutDefinition
  • Theme integration — Follows BeepThemesManager global theme; explicit colour API
  • Clip region optimization — Efficient Win32 rendering via dedicated painter adapter
  • DPI-aware — All layout calculations respect the host form’s DPI scale

Panel States

Each panel can be in one of four primary states, matching Visual Studio’s docking model:

  • Docked — The panel occupies a region of the form (Left, Right, Top, Bottom, Fill). Splitters separate docked panels.
  • Tabbed — Multiple panels share the same dockspace as stacked tabs. Click a tab to activate.
  • Floating — The panel is detached from the host form in a separate tool window. Drag the title bar to move.
  • Auto-Hidden — The panel collapses to a slim edge strip. Hover to peek (slide out).

Transitions between states are managed by the manager. Cancelable request events let the application veto state changes (e.g., prevent closing a document with unsaved changes).

// Example: prevent closing a panel with unsaved data
manager.PageCloseRequest += (s, e) =>
{
    if (e.PanelKey == "editor-panel" && HasUnsavedChanges(e.PanelKey))
    {
        e.Cancel = true;
        MessageBox.Show("Save changes before closing.");
    }
};

Theming

The docking manager supports three theming modes:

// 1. Follow BeepThemesManager globally
manager.Theme = "DarkTheme";
// Docking chrome automatically updates when BeepThemesManager.CurrentTheme changes.

// 2. Apply explicit colours (independent of theme system)
manager.ApplyTheme(
    background: Color.FromArgb(30, 30, 30),
    foreground: Color.White,
    border: Color.FromArgb(60, 60, 60),
    hover: Color.FromArgb(80, 80, 80),
    accent: Color.DodgerBlue
);

// 3. Apply a Beep theme object directly
var theme = BeepThemesManager.GetTheme("FluentLight");
manager.ApplyTheme(theme);

The Style property (BeepControlStyle) controls the shape of tabs and captions independent of colours, propagating to every docking surface.

Layout Persistence

Save and restore the entire docking layout using DockLayoutDefinition:

// Save layout
var definition = manager.LayoutTree.ToDefinition();
string json = JsonConvert.SerializeObject(definition, Formatting.Indented);
File.WriteAllText("layout.json", json);

// Restore layout
string json = File.ReadAllText("layout.json");
var definition = JsonConvert.DeserializeObject<DockLayoutDefinition>(json);
manager.LoadLayout(definition);

The definition preserves panel keys, titles, content types, docking positions, tab order, split ratios, and auto-hide state.

Examples

Basic Setup

var manager = new BeepDockingManager();

// Attach to form in Load event
private void MainForm_Load(object sender, EventArgs e)
{
    dockingManager.ManageControl(this);

    dockingManager.AddPanel("solution", "Solution Explorer",
        DockPosition.Right, new TreeView { Dock = DockStyle.Fill });

    dockingManager.AddPanel("output", "Output",
        DockPosition.Bottom, new TextBox { Dock = DockStyle.Fill, Multiline = true });

    dockingManager.AddPanel("editor1", "Main.cs",
        DockPosition.Fill, new RichTextBox { Dock = DockStyle.Fill });
}

Float and Re-dock

// Float a panel
manager.FloatPanel("output", new Rectangle(100, 100, 600, 400));

// Re-dock it to the right
manager.DockFloatingPanel("output", DockPosition.Right);

Themed Docking

// Sync with global theme changes
BeepThemesManager.Instance.PropertyChanged += (s, e) =>
{
    if (e.PropertyName == nameof(BeepThemesManager.CurrentTheme))
        dockingManager.ApplyTheme();
};
