BeepFormGenerator

A static utility class that retrofits standard WinForms Form instances with modern Beep-style chrome: borderless window, rounded corners, theme support, drag-to-move, edge resizing, and state preservation.

Overview

BeepFormGenerator is a public static utility class that applies a modern, borderless chrome to any standard System.Windows.Forms.Form. It modifies the form's FormBorderStyle to None, applies 20px rounded corners via the Win32 GDI CreateRoundRectRgn API, sets a BorderRadius-sized padding, and attaches mouse event handlers for custom window dragging and bottom-right corner resizing.

The class is designed to work alongside the full BeepiFormPro framework but can also be used independently. It saves the original form state before modification via the private FormState class, enabling clean restoration via RemoveBeepForm. Theme colors are applied through ApplyTheme using the IBeepTheme interface.

Unlike the full BeepiFormPro which inherits from Form and provides its own caption bar and window controls, BeepFormGenerator is a lightweight transformation layer — it takes an existing Form and modifies its appearance in-place.

Key Features

Borderless Chrome

Sets FormBorderStyle = None and applies 20px rounded corners using CreateRoundRectRgn.

Rounded Corners

GDI CreateRoundRectRgn creates a round-rect window region. Re-applied on every Resize for consistent appearance.

Edge Resizing

Bottom-right corner resize via mouse handlers: MouseDown/MouseMove/MouseUp. Cursor changes to SizeNWSE in the resize zone (within BorderRadius pixels of the bottom/right edge).

Drag-to-Move

The AttachDragMove method (placeholder for title-bar dragging via ReleaseCapture+SendMessage) is wired — the actual drag logic is provided by the caption bar in BeepiFormPro.

Theme Application

ApplyTheme sets form.BackColor from theme.BackgroundColor. Commented-out references show extension points for button, label, and title bar colors.

State Save / Restore

Original form state (border style, padding, colors, controls) is saved in a static Dictionary. RemoveBeepForm fully restores it.

Static Methods

MethodSignatureDescription
ApplyBeepForm public static void ApplyBeepForm(Form form, IBeepTheme theme = null) Transforms form into a borderless, rounded-corner window. Execution order:
1. Saves original state via FormState (if not already saved).
2. Sets FormBorderStyle = None.
3. Sets Padding = new Padding(BorderRadius) (20px).
4. Calls ApplyRoundedCorners(form) to set the window region.
5. Wires Resize to re-apply rounded corners.
6. Calls ApplyTheme(form, theme) if theme is non-null.
7. Calls AttachDragMove(form) (currently a no-op placeholder).
8. Wires MouseDown, MouseMove, MouseUp for resize support.
RemoveBeepForm public static void RemoveBeepForm(Form form) Restores form to its original WinForms state. If the form was saved in originalFormStates:
1. Restores FormBorderStyle, Padding, BackColor, ForeColor.
2. Sets form.Region = null to remove rounded corners.
3. Unhooks MouseDown, MouseMove, MouseUp, and the two Resize delegates.
4. Resets cursor to Cursors.Default.
5. Removes the entry from originalFormStates.
ApplyTheme public static void ApplyTheme(Form form, IBeepTheme theme) Applies theme colors. Sets form.BackColor = theme.BackgroundColor. Also contains commented-out code for setting button, label, and title bar colors — these are handled by the full BeepiFormPro framework instead.

Private Methods

MethodDescription
ApplyRoundedCorners(Form)Calls CreateRoundRectRgn(0, 0, form.Width, form.Height, 20, 20) via [DllImport("gdi32.dll")] and assigns the result to form.Region. This creates a 20px elliptical corner clipping region.
AttachDragMove(Form)Placeholder — contains commented-out code that would wire ReleaseCapture() + SendMessage(WM_SYSCOMMAND, SC_MOVE) for title-bar drag. In practice, dragging is handled externally by BeepiFormPro.
BeepForm_MouseDown(object, MouseEventArgs)Checks if the click is within the bottom/right BorderRadius zone. If so, sets isResizing = true and records lastMousePosition.
BeepForm_MouseMove(object, MouseEventArgs)During resize: adds delta to form.Width and form.Height, clamped to form.MinimumSize. Otherwise: sets cursor to SizeNWSE in the resize zone, Default elsewhere.
BeepForm_MouseUp(object, MouseEventArgs)Sets isResizing = false to stop resize mode.
RepositionControls(Form)Currently a no-op — contains commented-out logic for repositioning close/maximize/minimize buttons within a title panel. Handled by BeepiFormPro instead.

FormState & State Management

Before modifying a form, ApplyBeepForm saves its original state in a private nested class:

PropertyTypeDescription
FormBorderStyleFormBorderStyleThe original border style (e.g., Sizable, FixedDialog).
PaddingPaddingThe original form padding.
BackColorColorThe original background color.
ForeColorColorThe original foreground color.
OriginalControlsControl[]A snapshot copy of form.Controls at the time of application.

States are stored in private static readonly Dictionary<Form, FormState> originalFormStates. The dictionary keyed by form reference ensures that calling ApplyBeepForm twice on the same form does not overwrite the original state. RemoveBeepForm restores all saved properties and removes the entry.

Win32 Interop

The class uses three P/Invoke imports:

FunctionDLLPurpose
ReleaseCapture() user32.dll Releases mouse capture from the current window. Used as a prerequisite for SendMessage(SC_MOVE) to enable window dragging.
SendMessage(hWnd, Msg, wParam, lParam) user32.dll Sends 0x112 (WM_SYSCOMMAND) with 0xf012 (SC_MOVE) to initiate window move via the title bar. Currently commented out in AttachDragMove.
CreateRoundRectRgn(nLeft, nTop, nRight, nBottom, nWidthEllipse, nHeightEllipse) gdi32.dll Creates a GDI region with rounded rectangular corners. The result is assigned to form.Region to clip the window to a rounded shape with 20px corner radius.

Resize & Drag Logic

The resize behavior is custom-implemented through three mouse event handlers rather than relying on WM_NCHITTEST in WndProc:

  • Activation: In BeepForm_MouseDown, resize mode activates when the left mouse button is pressed within BorderRadius (20px) of the form's right or bottom edge (e.Location.X >= form.Width - BorderRadius || e.Location.Y >= form.Height - BorderRadius).
  • Resize: In BeepForm_MouseMove, while isResizing, the delta between the current and last mouse position is added to form.Width and form.Height, bounded by form.MinimumSize.
  • Deactivation: In BeepForm_MouseUp, isResizing is set to false.
  • Cursor Feedback: When not resizing, the cursor changes to Cursors.SizeNWSE when the mouse is in the resize zone, or Cursors.Default otherwise.

Theming via ApplyTheme

ApplyTheme(Form form, IBeepTheme theme) applies a minimal theme:

  • Sets form.BackColor = theme.BackgroundColor.

The commented-out code shows the intended (but unused from BeepFormGenerator) pattern for theme propagation to window chrome controls: close button, maximize button, minimize button, and title label. These are handled by the BeepiFormPro class and its interaction manager instead.

Examples

Example 1: Apply BeepForm Chrome

Transform a Standard Form
public partial class MyForm : Form
{
    public MyForm()
    {
        InitializeComponent();

        // Apply Beep-style chrome: rounded corners, borderless, resize
        BeepFormGenerator.ApplyBeepForm(this);

        // Optionally apply a theme
        var theme = BeepThemesManager.GetTheme("Dark");
        BeepFormGenerator.ApplyTheme(this, theme);
    }
}

Example 2: Restore Original Form State

Undo BeepForm Transformations
// Revert a form to its original WinForms appearance
BeepFormGenerator.RemoveBeepForm(this);

// After removal:
// - FormBorderStyle is restored
// - Padding, BackColor, ForeColor are restored
// - Rounded corners are removed (Region = null)
// - Mouse resize handlers are unhooked

Example 3: Toggle BeepForm Chrome

Switch Between Native and Beep Chrome
private bool _isBeepForm = false;

private void ToggleBeepChrome()
{
    if (_isBeepForm)
    {
        BeepFormGenerator.RemoveBeepForm(this);
        _isBeepForm = false;
    }
    else
    {
        BeepFormGenerator.ApplyBeepForm(this);
        _isBeepForm = true;
    }
}

Architecture

BeepFormGenerator.cs Contains two types:
BeepFormGenerator — static class with transform/restore/theme logic
FormState — private nested class storing original form properties
BeepiFormPro (Forms/ModernForm) The principled, full-featured modern form class. Uses BeepFormGenerator concepts internally but adds a complete caption bar, min/max/close buttons, resizing, and theming.
BeepiFormProInteractionManager Handles resize hit-testing and mouse interactions for BeepiFormPro, duplicating some of BeepFormGenerator's resize logic in a more polished way.
BeepiFormProHitAreaManager Manages the caption bar hit areas for dragging, double-click maximize, and system menu in BeepiFormPro.
