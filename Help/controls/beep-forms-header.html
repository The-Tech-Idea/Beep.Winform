Overview

BeepFormsHeader is a visual header bar that displays the form title and active-context information from a bound BeepForms coordinator. It automatically listens for block, forms-manager, and view-state changes and refreshes its display accordingly.

Namespace: TheTechIdea.Beep.Winform.Controls.Integrated.Forms
Assembly: TheTechIdea.Beep.Winform.Controls.Integrated.dll
Inheritance: BaseControl → BeepFormsHeader
Designer: BeepFormsHeaderDesigner

Uses a two-row TableLayoutPanel with a BeepLabel for the title and an optional context line that shows the active block name, query-mode indicator, and dirty-state colored by severity.

Key Features

  • Auto-bind to BeepForms host — Automatically discovers a nearby BeepForms in the control hierarchy when AutoBindFormsHost is true
  • Active block display — Shows the current active block name in the context line
  • State summary — Displays query mode and dirty-state indicators with color coding (orange=dirty, blue=query, green=active block, gray=idle)
  • Auto-height adjustment — Collapses the context row when it is empty, reducing overall height
  • Event-driven refresh — Listens to ActiveBlockChanged, FormsManagerChanged, and ViewStateChanged for automatic updates

Theming

BeepFormsHeader sets UseThemeColors = true and delegates theme colors to its child BeepLabel controls. The context label color is resolved dynamically based on view state: DarkOrange for dirty, SteelBlue for query mode, ForestGreen for active block, and DimGray for idle. Font sizes use BeepFontManager.TitleFont and BeepFontManager.StatusBarFont.

Examples

// Add a BeepFormsHeader that auto-binds to the first BeepForms on the form
var header = new BeepFormsHeader
{
    Dock = DockStyle.Top,
    ShowActiveBlock = true,
    ShowStateSummary = true
};
this.Controls.Add(header);

// Explicitly bind to a specific BeepForms instance
var beepForms = new BeepForms { FormName = "CustomerForm" };
var header2 = new BeepFormsHeader
{
    FormsHost = beepForms,
    AutoBindFormsHost = false
};

Architecture

The control contains an internal TableLayoutPanel with two rows: the title row (28px fixed) and the context row (18px when visible, 0px when hidden). When the context label is invisible, the overall height collapses from 60px to 42px. Host binding is resolved via BeepFormsHostResolver.Find() which walks the parent hierarchy looking for a BeepForms instance. The display text is resolved through BeepFormsDisplayTextResolver.ResolveTitle() and ResolveContext() helpers.
