Overview

BeepFormsQueryShelf is a composite control providing query-mode actions for BeepForms. It displays an optional caption line showing the current query target, plus “Enter Query” and “Execute Query” buttons that call EnterQueryAsync() and ExecuteQueryAsync() on the bound host.

Key Features

  • Enter Query — Enables query mode on the active block; button is enabled when an active block exists
  • Execute Query — Executes the current query on the active block; button is only enabled when already in query mode
  • Context caption — Optional caption line shows which block the query targets, with color-coded state (green=ready, blue=in query, orange=no block)
  • Caption mode control — QueryCaptionMode enum controls how much host context is displayed
  • Configurable buttons — QueryButtons flags enum selects which buttons are visible

Theming

BeepFormsQueryShelf uses UseThemeColors = true and passes the Theme to all child BeepButton and BeepLabel controls. The caption label color is resolved by ResolveCaptionColor(): DimGray when no host is bound, DarkOrange when no active block is set, SteelBlue when in query mode, and ForestGreen when ready. Font uses BeepFontManager.StatusBarFont.

Examples

// Full query shelf with context caption
var queryShelf = new BeepFormsQueryShelf
{
    Dock = DockStyle.Top,
    QueryButtons = BeepFormsQueryShelfButtons.All,
    ShowQueryContextCaption = true,
    QueryCaptionMode = BeepFormsQueryShelfCaptionMode.TargetPlusMode,
    FormsHost = myBeepForms
};

// Minimal query shelf with only Execute Query, no caption
var queryShelf2 = new BeepFormsQueryShelf
{
    QueryButtons = BeepFormsQueryShelfButtons.ExecuteQuery,
    ShowQueryContextCaption = false
};
this.Controls.Add(queryShelf2);

Architecture

The control uses a two-row TableLayoutPanel: a 16px caption row and a 28px button row. RefreshQueryShelf() rebuilds the buttons from the QueryButtons flags. When the caption is hidden, the layout collapses from 54px to 36px. Button enablement is state-aware: “Enter Query” requires an active block; “Execute Query” requires the active block to already be in query mode (viewState.IsQueryMode). The command panel is hidden when no buttons are configured.
