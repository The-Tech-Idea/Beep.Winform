Overview

BeepFormsToolbar is a composite toolbar combining savepoint management and alert preset actions into two popup buttons. It binds to a BeepForms host and provides granular control over which actions are available through multiple flags enums.

Namespace: TheTechIdea.Beep.Winform.Controls.Integrated.Forms
Assembly: TheTechIdea.Beep.Winform.Controls.Integrated.dll
Inheritance: BaseControl → BeepFormsToolbar (partial class)
Designer: BeepFormsToolbarDesigner

Defined as a partial class spanning two files. The main file handles layout and button construction; the *.Actions.cs file handles click routing, savepoint dialogs, alert presets, and async workflow orchestration.

Key Features

  • Savepoint popup — Popup button with configurable actions: Capture, List, Rollback, Release, and Release All savepoints
  • Alert presets popup — Popup button for predefined alert workflows: Info, Caution, Stop, and Question (confirm dialog)
  • Flexible button ordering — ShellToolbarOrder controls whether savepoints or alerts appear first
  • Captions configurable — Both popup button captions are customizable via SavepointToolbarCaption and AlertToolbarCaption
  • Context-aware enablement — Savepoint button enables only when a block context exists in the FormsManager; alert button enables when the FormsManager has an alert provider

Theming

BeepFormsToolbar uses UseThemeColors = true and passes the current Theme to all BeepButton children. Both popup buttons are created via CreateToolbarPopupButton() which sets PopupMode = true, disables shadows, and uses theme colors. Button widths are measured via GetToolbarPopupButtonWidth() with a minimum of 128px (savepoints) and 112px (alerts).

Examples

// Full toolbar with both savepoints and alerts
var toolbar = new BeepFormsToolbar
{
    Dock = DockStyle.Top,
    ShellToolbarButtons = BeepFormsShellToolbarButtons.All,
    SavepointToolbarActions = BeepFormsSavepointToolbarActions.All,
    AlertToolbarPresets = BeepFormsAlertToolbarPresets.All,
    SavepointToolbarCaption = "Savepoints",
    AlertToolbarCaption = "Alerts",
    FormsHost = myBeepForms
};

// Minimal toolbar: alerts-only with just Info and Question presets
var alertsOnly = new BeepFormsToolbar
{
    ShellToolbarButtons = BeepFormsShellToolbarButtons.Alerts,
    AlertToolbarPresets = BeepFormsAlertToolbarPresets.Info | BeepFormsAlertToolbarPresets.Question,
    AlertToolbarCaption = "Notify"
};
this.Controls.Add(alertsOnly);

Architecture

The toolbar wraps a FlowLayoutPanel rebuilt by InitializeCommandStrip() on any configuration change. Button ordering follows ShellToolbarOrder. The savepoint popup builds items from SavepointToolbarActions flags (Capture, List, Rollback, Release, ReleaseAll) and the alert popup from AlertToolbarPresets (Info, Caution, Stop, Question). Click handlers in the Actions.cs partial class use BeepFormsDialogService for user interaction dialogs (text prompt, picker, list viewer) and delegate savepoint operations to the bound host via CreateSavepoint, RollbackToSavepointAsync, ReleaseSavepoint, and ReleaseAllSavepoints.
