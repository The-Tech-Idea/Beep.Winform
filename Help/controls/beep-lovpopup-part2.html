Async Data Loading

Manual Loading

BeginLoading / EndLoading
popup.BeginLoading();
var items = await FetchItemsAsync();
popup.EndLoading(items);

LoadItemsAsync (Recommended)

Single Async Call with Cancellation
await popup.LoadItemsAsync(async (token) =>
{
    var data = await db.SearchAsync(query, token);
    return data.Select(d => new SimpleItem
    {
        Text = d.Name,
        Value = d.Id
    }).ToList();
}, preloadSearch: "cust");

Key behaviors: auto-cancels previous loads, silently swallows OperationCanceledException, shows errors in loading label, checks IsDisposed/IsHandleCreated before UI updates.

Recent Items System

The recent items system persists the last 5 selections in a Queue<SimpleItem> (oldest-first). When the user accepts a selection, PushRecent removes any duplicate (by Value equality) and adds the item. If the queue exceeds 5, the oldest is evicted.

The RecentItems property enables cross-cycle persistence:

Persisting Recent Items
var savedItems = popup.RecentItems.ToList();   // save
popup.RecentItems = savedItems;               // restore

Theming

ApplyLovTheme() applies CurrentTheme to every visual element:

  • Form: BackColor = PanelBackColor, ForeColor.
  • Header: GridHeaderBackColor (falls back to PanelBackColor).
  • Footer & Recent Panel: PanelBackColor.
  • Count Label: SecondaryTextColor foreground, transparent background.
  • BeepControls: Each child gets Theme = LovTheme, UseThemeColors, ApplyTheme().
  • Grid: _grid.Theme = LovTheme, _grid.ApplyTheme().
  • Loading Overlay: Semi-transparent (alpha 220) PanelBackColor.
  • Border: CurrentTheme?.BorderColor via OnPaint.

Examples

Example 1: Basic Popup from a TextBox

Show LOV Below a Control
private BeepLovPopup _popup = new BeepLovPopup();

private void ShowLov(TextBox owner)
{
    var items = new List<SimpleItem>
    {
        new() { Value = 1, Text = "United States" },
        new() { Value = 2, Text = "Canada" },
        new() { Value = 3, Text = "Mexico" },
    };

    _popup.LovTheme = "Dark";
    _popup.UseThemeColors = true;
    _popup.CurrentTheme = BeepThemesManager.GetTheme("Dark");
    _popup.ItemAccepted += (s, item) =>
    {
        owner.Text = item.Text;
        owner.Tag = item.Value;
    };

    var screenPoint = owner.Parent.PointToScreen(owner.Location);
    _popup.ShowAt(items,
        new Point(screenPoint.X, screenPoint.Y + owner.Height),
        owner.Width);
}

Selection Pipeline

Full Data Flow
ShowAt(items, origin, width, preloadSearch)
  → _allItems = items
  → _searchBox.Text = preloadSearch
  → FilterItems(search) → _filteredItems
  → ConfigureGridColumns() → grid columns
  → RebindGrid() → grid.DataSource
  → UpdateRecentPanel() → chip strip
  → Calc size + position + ApplyLovTheme()
  → Show() + _searchBox.Focus()

User types: SearchBox_TextChanged → FilterItems → RebindGrid → UpdateCount
                                → SearchChanged event (server-side)

Accepts: Enter / F9 / DoubleClick / OK / Recent chip → Accept()
  → PushRecent(item) → SelectedItem = item → ItemAccepted → Hide()

Cancels: Escape / Cancel / Close / OnDeactivate → Cancel()
  → Cancelled → Hide()
