Overview

BeepNavBar is a modern, lightweight navigation bar control for Windows Forms. It displays a collection of SimpleItem objects as clickable navigation items with icons and text. The control supports both horizontal (icon-above-text) and vertical (icon-beside-text) orientations with DPI-aware automatic item sizing to prevent text clipping.

BeepNavBar serves as the base class for TopNavBar and BottomNavBar and follows the same painter-based architecture as BeepSideBar. It inherits from BaseControl and uses double-buffered rendering with ResizeRedraw for smooth layout updates.

The control uses a MenuItemCollectionEditor for design-time item editing and safely disables expensive layout calculations during design-time to prevent flicker.

Key Features

Dual Orientation

Horizontal mode (icon above text, default) and Vertical mode (icon left, text right) via NavBarOrientation enum

Automatic Item Sizing

DPI-aware RecalculateItemSizes() measures text and icons to compute optimal item dimensions, preventing text clipping

Painter-Based Rendering

Follows the same architecturally consistent painter pattern used by BeepSideBar with double-buffered, user-painted output

Layout Caching

Dirty-flag tracking (_layoutDirty, _itemSizesDirty) prevents redundant hit-area and size recalculations

Design-Time Safety

IsDesignModeSafe property detects all design-mode scenarios (LicenseManager, DesignMode, Site.DesignMode) to skip expensive runtime operations

Theme Integration

Overrides ApplyTheme() to synchronize fonts from the theme's NavigationUnSelectedFont property

Visual Customization

Configurable accent color, shadow effects, corner radius, and per-item width/height with theme-color toggle

Collection Editor

MenuItemCollectionEditor provides a rich design-time editor for managing the BindingList<SimpleItem> items collection

Theming

BeepNavBar integrates with the Beep theming system via its ApplyTheme() override. When a theme is active, the control's text font is derived from the theme's NavigationUnSelectedFont property through BeepFontManager.ToFont(). If no theme is set, it falls back to BeepFontManager.DefaultFont.

The UseThemeColors property (default true) lets you choose between theme-driven colors and a manually specified AccentColor. Derived painter classes are responsible for querying these properties to resolve the effective color for highlights, selection banners, and focus indicators.

Theme Integration

var navbar = new BeepNavBar
{
    UseThemeColors = true,  // Follow theme accent automatically
    EnableShadow = true,
    CornerRadius = 10
};

// ThemeChanged handlers should call:
navbar.ApplyTheme();

Custom Accent Override

var navbar = new BeepNavBar
{
    UseThemeColors = false,
    AccentColor = Color.FromArgb(255, 87, 34)  // Deep orange accent
};

Examples

Basic Horizontal Navigation Bar

var navbar = new BeepNavBar
{
    Dock = DockStyle.Top,
    Orientation = NavBarOrientation.Horizontal,
    ItemWidth = 80,
    ItemHeight = 56,
    AccentColor = Color.FromArgb(0, 120, 215),
    UseThemeColors = true,
    CornerRadius = 8
};

navbar.Items.Add(new SimpleItem { Text = "Home", ImagePath = "home.svg" });
navbar.Items.Add(new SimpleItem { Text = "Explore", ImagePath = "explore.svg" });
navbar.Items.Add(new SimpleItem { Text = "Notifications", ImagePath = "bell.svg" });
navbar.Items.Add(new SimpleItem { Text = "Profile", ImagePath = "user.svg" });

navbar.ItemClicked += item =>
{
    Console.WriteLine($"Navigating to: {item?.Text}");
};

this.Controls.Add(navbar);

Vertical Side Navigation

var sidebar = new BeepNavBar
{
    Dock = DockStyle.Left,
    Width = 200,
    Orientation = NavBarOrientation.Vertical,
    ItemWidth = 180,
    ItemHeight = 44,
    CornerRadius = 6
};

// Items display icon-left, text-right in vertical mode
sidebar.Items.Add(new SimpleItem { Text = "Dashboard", ImagePath = "dashboard.svg" });
sidebar.Items.Add(new SimpleItem { Text = "Analytics", ImagePath = "chart.svg" });
sidebar.Items.Add(new SimpleItem { Text = "Messages", ImagePath = "envelope.svg" });
sidebar.Items.Add(new SimpleItem { Text = "Calendar", ImagePath = "calendar.svg" });
sidebar.Items.Add(new SimpleItem { Text = "Settings", ImagePath = "gear.svg" });

sidebar.ItemClicked += item =>
{
    // Switch views based on selected item
    NavigateToView(item?.Text);
};

sidebar.PropertyChanged += (s, e) =>
{
    if (e.PropertyName == nameof(BeepNavBar.SelectedItem))
    {
        Console.WriteLine($"Selection changed");
    }
};

Programmatic Item Management

// Add items
navbar.Items.Add(new SimpleItem { Text = "New Tab", ImagePath = "plus.svg", GuidId = "tab-1" });

// Move items
var item = navbar.Items[0];
navbar.Items.RemoveAt(0);
navbar.Items.Insert(2, item);

// Replace entire collection
navbar.Items = new BindingList
{
    new SimpleItem { Text = "A", ImagePath = "a.svg" },
    new SimpleItem { Text = "B", ImagePath = "b.svg" },
    new SimpleItem { Text = "C", ImagePath = "c.svg" }
};

// Items property setter automatically re-subscribes ListChanged and triggers full layout refresh

Dynamic Orientation Switch

// Toggle between horizontal (top) and vertical (left) mode
private void ToggleNavBarMode(BeepNavBar navbar)
{
    if (navbar.Orientation == NavBarOrientation.Horizontal)
    {
        navbar.Orientation = NavBarOrientation.Vertical;
        navbar.Dock = DockStyle.Left;
        navbar.Width = 200;
    }
    else
    {
        navbar.Orientation = NavBarOrientation.Horizontal;
        navbar.Dock = DockStyle.Top;
        navbar.Height = navbar.ItemHeight;
    }
}

Handling Clicks with Selection

navbar.ItemClicked += clickedItem =>
{
    // Clear selection on all items visually (if needed by your painter)
    foreach (var item in navbar.Items)
    {
        // Derived controls may update item states here
    }

    // The SelectedItem is already set by OnItemClicked before this fires
    var selected = navbar.SelectedItem;
    if (selected != null)
    {
        Console.WriteLine($"Active nav: {selected.Text} (ID: {selected.GuidId})");
    }
};

Architecture

BeepNavBar uses a single-file partial class design (one .cs file with a .Designer.cs companion generated by the WinForms designer). It does not employ the multi-file partial organization of BeepDock, instead consolidating all logic within the main file for simplicity:
