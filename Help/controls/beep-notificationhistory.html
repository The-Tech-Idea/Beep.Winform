Overview

BeepNotificationHistory is a customizable history panel that stores and displays dismissed notifications. It provides header controls for search, type filtering, and read-status filtering, a scrollable list with custom-painted notification items color-coded by type, and bulk actions like "Clear" and "Mark All Read."

Namespace: TheTechIdea.Beep.Winform.Controls.Notifications  |  Inheritance: Component → Control → BaseControl → BeepNotificationHistory

Toolbox: Category("Beep Controls"), Description("Notification history panel with filtering and search")

Key Features

  • History storage — Stores up to 100 dismissed notifications (configurable via MaxHistorySize) with automatic trimming
  • Type filtering — Dropdown filter for Info, Success, Warning, Error, and System notification types
  • Read-status filtering — Filter by Read, Unread, or All status
  • Search with debounce — 250ms debounced text search across notification titles and messages
  • Custom painting — Each history item is drawn with type-specific background tinting, bold title, muted message, and relative timestamp
  • Bulk actions — "Clear" removes all history; "Mark All Read" sets every item as read
  • Dpi-aware — All dimensions, padding, and item heights scale with DPI via DpiScalingHelper
  • Click-to-detail — HistoryItemClicked event fires when a user clicks a history item row

Theming

BeepNotificationHistory uses Material3 style by default (ControlStyle = BeepControlStyle.Material3) with rounded corners, shadow, and a 1px border. The ApplyTheme() override calls base.ApplyTheme() and then sets the internal _textFont from _currentTheme.BodyMedium via BeepFontManager.ToFont(). History items are drawn with type-specific translucent background colors (Success = green, Warning = orange, Error = red, Info = blue, System = gray).

Examples

Adding Notifications to History

var history = new BeepNotificationHistory {
    MaxHistorySize = 50,
    ItemHeight = 64
};

// When a notification is dismissed, add it to history
history.AddNotification(new NotificationData {
    Title = "Export Complete",
    Message = "CSV file exported successfully.",
    Type = NotificationType.Success,
    IsRead = false
});

history.AddNotification(new NotificationData {
    Title = "Connection Lost",
    Message = "Database connection timed out.",
    Type = NotificationType.Error,
    IsRead = false
});

Handling Clicks

history.HistoryItemClicked += (s, item) => {
    item.Data.IsRead = true;
    ShowNotificationDetail(item.Data);
};

Programmatic Setup

var history = new BeepNotificationHistory {
    Dock = DockStyle.Right,
    Width = 350
};
this.Controls.Add(history);

Architecture

BeepNotificationHistory extends BaseControl and is composed of three layered child controls: a _headerPanel (DockTop, 100px) containing labels, textboxes, combo boxes, and action buttons; a _listPanel (DockFill) with AutoScroll = false and custom Paint handling; and a VScrollBar (DockRight) for virtual scrolling. The internal _history list stores NotificationHistoryItem objects. The list panel paints items using GDI+ graphics with anti-aliased text and type-specific translucent backgrounds. Mouse wheel and click events are handled directly on the list panel for smooth interaction.
