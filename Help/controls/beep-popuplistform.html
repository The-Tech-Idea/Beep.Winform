BeepPopupListForm

Lightweight popup list form for displaying selectable item lists with search filtering and extendable context menus

Overview

BeepPopupListForm is a popup form that hosts an internal BeepListBox for displaying a list of SimpleItem entries. It serves as the dropdown for BeepComboBox and as a standalone context-menu-style popup. Supports search filtering, child cascading popups for hierarchical items, and extension method invocation.

Namespace: TheTechIdea.Beep.Winform.Controls  |  Inheritance: Form → BeepPopupForm → BeepPopupListForm

Key Features

  • Popup list display — Renders a filtered BeepListBox inside a borderless popup form
  • Search filtering — Filter(string) method for live text filtering of list items
  • Cascading child popups — Items with children open sub-popups positioned to the right
  • Extension method support — RunFunctionFromExtensions delegate for invoking global methods from list items
  • Auto-sizing — Calculates optimal width and height based on item text and count
  • Customizable font — TextFont property overrides the theme font for list items
  • Configurable title bar — ShowTitle toggles a title caption on the popup
  • Positional popup — ShowPopup() overloads support trigger-control, anchor-point, and offset-based positioning

Theming

BeepPopupListForm.ApplyTheme() applies the active theme to both the popup form and its internal BeepListBox. Theme tokens consumed include ListBackColor, ListForeColor, and ListBorderColor. The internal listbox is configured with no borders or shadows, as the popup form handles its own border drawing through the FormPainterMetrics system.

Examples

Basic Popup List

var items = new List<SimpleItem> {
    new SimpleItem { Text = "Option A", Value = 1 },
    new SimpleItem { Text = "Option B", Value = 2 },
    new SimpleItem { Text = "Option C", Value = 3 }
};

// Show popup next to a button
var popup = new BeepPopupListForm(items);
popup.ItemSelected += (s, item) => Console.WriteLine($"Selected: {item.Text}");
popup.ShowPopup("Choose Option", myButton, BeepPopupFormPosition.Bottom);

Hierarchical Cascading Popups

var parentItem = new SimpleItem {
    Text = "Settings",
    Children = new BindingList<SimpleItem> {
        new SimpleItem { Text = "General" },
        new SimpleItem { Text = "Advanced" }
    }
};

var popup = new BeepPopupListForm(new List<SimpleItem> { parentItem });
// Clicking "Settings" automatically opens a child BeepPopupListForm to the right
popup.ShowPopup("Menu", myButton, BeepPopupFormPosition.Bottom);

Filtering the List

var popup = new BeepPopupListForm(allItems);
popup.Filter("sea"); // Only items containing "sea" are shown
popup.ShowPopup("Search Results", myTextBox, BeepPopupFormPosition.Bottom);

Architecture

BeepPopupListForm is a partial class extending BeepPopupForm. It contains an auto-generated BeepListBox (_beepListBox) created in the .Designer.cs partial file. The form is configured as borderless (FormBorderStyle.None) with double-buffering enabled to prevent flicker. Border and shadow calculations are performed via FormPainterMetrics and StyleShadows, with the listbox positioned inside the form's client area offset by padding to prevent overlap with painted borders.

The control is declared as public partial class BeepPopupListForm : BeepPopupForm in TheTechIdea.Beep.Winform.Controls namespace.
