Overview

BeepProjectCard (class: BeepProjectTaskCard) is a visual project management card designed for Kanban-style boards. It inherits from BaseControl and resides in the TheTechIdea.Beep.Winform.Controls.ProjectCards namespace. The control renders status/priority badges with icons, project title and description, team avatars, task metrics, a progress bar, and View/Update action buttons. It supports compact mode and full theme awareness via ApplyTheme().

Key Features

  • Status and priority badges with auto-color mapping (Not Started, In Progress, Completed, On Hold, Cancelled / Low, Medium, High, Critical)
  • Team avatars with overflow indicator rendered via BeepImage
  • Progress bar with percentage label
  • Task metrics (completed/total tasks, hours spent/estimated)
  • Due date display with overdue highlighting
  • View Details and Update action buttons
  • Compact mode for smaller card sizes
  • Gradient background with rounded corners and anti-aliased drawing
  • Full theme integration via ApplyTheme()
  • Selectable card (mouse events fire ProjectClicked and ActionButtonClicked)

Theming

The control fully responds to ApplyTheme(). It sets BackColor from _currentTheme.TaskCardBackColor, gradient from GradientStartColor / GradientEndColor, border from TaskCardBorderColor, and foreground from TaskCardForeColor. Child labels and images internally use _currentTheme properties for fonts (TaskCardTitleFont, TaskCardSubStyleStyle, TaskCardMetricTextStyle) and colors (TaskCardTitleForeColor, TaskCardMetricTextForeColor). Status and priority colors are determined by string-switch logic with fallback to SubLabelForColor.

Examples

// Basic project card
var card = new BeepProjectTaskCard
{
    ProjectTitle = "Website Redesign",
    ProjectDescription = "Complete overhaul of the company landing page",
    ProjectStatus = "In Progress",
    ProjectPriority = "High",
    ProgressValue = 65f,
    CompletedTasks = 8,
    TotalTasks = 15,
    HoursSpent = 42,
    EstimatedHours = 80,
    ProjectManager = "Alice Johnson",
    TeamCount = "5",
    DueDate = DateTime.Now.AddDays(14),
    CompactMode = false,
    Size = new Size(300, 350)
};

card.ActionButtonClicked += (s, e) =>
{
    switch (e.Action)
    {
        case "ViewDetails": OpenProjectView(e.ProjectTitle); break;
        case "Update": ShowStatusDialog(e.ProjectTitle); break;
    }
};

card.ProjectClicked += (s, e) =>
{
    NavigateToProject(e.ProjectTitle);
};

this.Controls.Add(card);

Architecture

BeepProjectTaskCard extends BaseControl and overrides DrawContent to render the card in distinct sections (header badges, content area, metrics, progress bar, action buttons). It uses reusable BeepImage and BeepLabel instances as drawing helpers. Team avatars are rendered as rounded images with a white border, and an overflow "+N" badge appears when avatars exceed the visible limit. The card is selectable (TabStop = true) and uses hit-testing in OnMouseDown to distinguish between View Details and Update button clicks. Status and priority colors are determined by case-insensitive string matching with predefined color palettes.
