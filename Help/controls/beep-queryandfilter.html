Overview

BeepQueryandFilter is a data-aware control that dynamically generates filter input fields from an EntityStructure definition. It creates appropriate input controls (TextBox, NumericUpDown, DateTimePicker, CheckBox) for each field, supports range queries for numeric and date types, and provides a submit button that returns a list of AppFilter objects.

Namespace: TheTechIdea.Beep.Winform.Controls  |  Inheritance: Component → Control → BaseControl → BeepQueryandFilter

Toolbox: Category("Data"), DisplayName("Beep Query and Filter")

Key Features

  • Automatic control generation — Creates label + input pairs for every field in an EntityStructure
  • Type-aware inputs — Generates DateTimePicker for dates, NumericUpDown for numbers, CheckBox for booleans, TextBox for everything else
  • Range query support — Numeric and date fields get both "from" and "to" inputs for range filtering
  • Submit event — Clicking the Submit button triggers SubmitClicked with the completed filter data
  • AppFilter extraction — GetFilters() reads all input values and returns typed AppFilter objects
  • Layout-managed — Uses a 3-column TableLayoutPanel (label, from-input, to-input) with auto-scroll

Methods

MethodSignatureDescription
CreateQueryControlsvoid CreateQueryControls(EntityStructure entityStructure)Clears all existing controls and generates a new set of label/input pairs from the entity structure fields.
GetFiltersList<AppFilter> GetFilters()Reads values from all input controls and returns a list of populated AppFilter objects with FilterValue (from) and FilterValue1 (to) set.

Examples

Generating Filter Controls

var queryFilter = new BeepQueryandFilter();

var entity = new EntityStructure {
    Fields = new List<EntityField> {
        new EntityField { FieldName = "ProductName", Fieldtype = "System.String" },
        new EntityField { FieldName = "Price", Fieldtype = "System.Decimal" },
        new EntityField { FieldName = "OrderDate", Fieldtype = "System.DateTime" },
        new EntityField { FieldName = "IsActive", Fieldtype = "System.Boolean" }
    }
};

// Creates: TextBox for ProductName, NumericUpDown (from/to) for Price,
// DateTimePicker (from/to) for OrderDate, CheckBox for IsActive
queryFilter.CreateQueryControls(entity);

Retrieving Filter Values

var queryFilter = new BeepQueryandFilter();
queryFilter.CreateQueryControls(myEntityStructure);

queryFilter.SubmitClicked += (s, e) => {
    List<AppFilter> filters = queryFilter.GetFilters();
    foreach (var filter in filters)
    {
        if (!string.IsNullOrEmpty(filter.FilterValue))
            Console.WriteLine($"{filter.FieldName}: {filter.FilterValue} - {filter.FilterValue1}");
    }
    // Apply filters to data source
    ExecuteFilteredQuery(filters);
};

Programmatic Layout Integration

// Dock in a panel for form integration
var queryFilter = new BeepQueryandFilter {
    Dock = DockStyle.Top,
    AutoSize = true
};
queryFilter.CreateQueryControls(myEntityStructure);
this.Controls.Add(queryFilter);

Architecture

BeepQueryandFilter extends BaseControl and is lightweight with no public properties beyond base control theming. It maintains an internal List<AppFilter> Filters collection and a 3-column TableLayoutPanel that hosts dynamically created controls. The Submit button is added at the bottom of the layout with a column span of 2. Each input control's Tag property stores a tuple (AppFilter, bool isFrom) for later value extraction in GetFilters().

The control is declared at TheTechIdea.Beep.Winform.Controls.BeepQueryandFilter with [ToolboxItem(true)] and [Category("Data")] for discoverability in the Visual Studio toolbox.
