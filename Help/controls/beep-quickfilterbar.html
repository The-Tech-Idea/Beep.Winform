Overview

BeepQuickFilterBar is a BaseControl-derived composite that emulates Excel's quick filter toolbar. It provides a column selector dropdown, a search text box, clear and advanced filter buttons, and a filter count badge. The control emits FilterChanged events as the user types or changes the column selector, and AdvancedFilterRequested when the ⚙ Advanced button is clicked.

The control is designed to be hosted immediately above or below a BeepGridPro. It communicates with the grid through events, never directly — the grid's parent form wires the events to the appropriate filtering logic. The FilterEventArgs class carries SearchText and ColumnName (nullable — null means "all columns").

The control also supports loading filter state from a FilterConfiguration object via SetFromConfiguration, making it bidirectional: the bar can both generate and receive filter criteria.

Key Features

Column Selector

A BeepComboBox populated with column names from the host grid. The first item is always "All Columns". Changing the selection fires FilterChanged.

Live Search

A BeepTextBox with placeholder "Type to filter...". Every keystroke fires FilterChanged with the current text and selected column.

Filter Count Badge

A BeepLabel shows "🔖 N filters" when ActiveFilterCount > 0. Automatically uses plural ("filter" vs "filters") and is hidden when count is zero.

Advanced Filter Gateway

The ⚙ Advanced button fires AdvancedFilterRequested. The parent form typically responds by opening BeepAdvancedFilterDialog.

Configuration Sync

SetFromConfiguration reads a FilterConfiguration and populates the search text and column selector — for single-criterion Contains filters.

Themed

Built with BaseControl + themed child controls. Shows borders and shadow by default (IsFrameless = false, ShowAllBorders = true, ShowShadow = true).

Properties

PropertyTypeAccessDefaultDescription
SearchText string Get / Set "" Current text in the search box. Setting this updates _searchTextBox.Text, which triggers FilterChanged.
SelectedColumn string Get / Set "All Columns" The currently selected column name. On get, returns the Name property of the selected SimpleItem, or "All Columns" if none selected. On set, finds the matching SimpleItem and sets SelectedItem on the combo.
AvailableColumns string[] Write-only — Sets the column names available in the dropdown. Clears and rebuilds the combo items list (always preserving "All Columns" as the first entry). Sets SelectedIndex = 0.
ActiveFilterCount int Get / Set 0 The number of active filters to display in the badge. Setting this calls UpdateFilterCountDisplay() which formats the badge text as "🔖 N filter(s)" and toggles visibility.

Events & FilterEventArgs

EventArgs TypeDescription
FilterChanged FilterEventArgs Fires when the search text changes or the column selector changes (_searchTextBox.TextChanged and _columnSelector.SelectedIndexChanged). The event args carry the current SearchText and ColumnName (where null means all columns).
AdvancedFilterRequested EventArgs Fires when the user clicks the ⚙ Advanced button. Does not carry filter data — the consumer should open a full filter dialog (e.g., BeepAdvancedFilterDialog) and combine its result with the current quick filter state.

Methods

Public Methods

MethodSignatureDescription
SetFromConfiguration public void SetFromConfiguration(FilterConfiguration config) Synchronizes the bar's state from an existing FilterConfiguration. If the config is null or has no criteria, calls ClearFilter(). If the config has exactly one criterion with FilterOperator.Contains, sets SelectedColumn and SearchText to match. In all cases, sets ActiveFilterCount from config.EnabledFilterCount.

Examples

Example 1: Basic Setup with a Grid

Attach Quick Filter Bar to BeepGridPro
var filterBar = new BeepQuickFilterBar
{
    Dock = DockStyle.Top
};

// Populate available columns from the grid
var columnNames = grid.ColumnConfigs
    .Where(c => c.Visible && !c.IsSelectionCheckBox)
    .Select(c => c.ColumnName)
    .ToArray();
filterBar.AvailableColumns = columnNames;

// React to filter changes
filterBar.FilterChanged += (s, e) =>
{
    if (string.IsNullOrWhiteSpace(e.SearchText))
    {
        grid.ClearFilters();
    }
    else
    {
        grid.ApplyQuickFilter(e.SearchText, e.ColumnName);
    }
    filterBar.ActiveFilterCount = string.IsNullOrWhiteSpace(e.SearchText) ? 0 : 1;
};

// Open advanced dialog
filterBar.AdvancedFilterRequested += (s, e) =>
{
    var dialog = new BeepAdvancedFilterDialog(grid.ColumnConfigs);
    if (dialog.ShowDialog() == DialogResult.OK && dialog.Result != null)
    {
        grid.ApplyFilterConfiguration(dialog.Result);
        filterBar.SetFromConfiguration(dialog.Result);
    }
    dialog.Dispose();
};

this.Controls.Add(filterBar);
this.Controls.Add(grid); // grid.Dock = Fill

Architecture

BeepQuickFilterBar.cs Contains two classes:
BeepQuickFilterBar — the quick filter toolbar

BaseControl (Base) Base class providing theming, borders, and shadow.
