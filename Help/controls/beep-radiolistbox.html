BeepRadioListBox

A combined control integrating BeepRadioGroup and BeepListBox with bidirectional selection synchronization and search

Overview

BeepRadioListBox combines a BeepRadioGroup (chip-style radio buttons) with a BeepListBox (detailed list view) in a single composite control. Both views share a single BindingList<SimpleItem> data source and maintain bidirectional selection synchronization — clicking an item in one view instantly selects it in the other.

An optional search box at the top filters the list in real time. The control's layout is configurable: radio above list, list above radio, radio-only, list-only, or compact mode.

Key Features

  • Dual-view selection — Chip-style radio group above a scrollable list box, both driven by the same item collection
  • Bidirectional sync — Selecting in the radio group updates the list box and vice versa, with infinite-loop prevention
  • Search/filter — BeepTextBox with search icon filters the list box as you type
  • Five layout modes — RadioAboveList, RadioBelowList, RadioOnly, ListOnly, Compact
  • Component visibility toggles — ShowSearch, ShowRadioGroup, ShowListBox, and ShowDivider control which parts render
  • Radio group styling — RadioStyle (Flat/Chip/Card/etc.) and RadioOrientation (Horizontal/Vertical)
  • List box styling — ListStyle delegates to BeepListBox.ListBoxType
  • Collection editor support — Items property is editable in the Visual Studio designer
  • Selection tracking — RadioListBoxSelectionChangedEventArgs reports the selected item and which control originated the change

Theming

The control fully respects the Beep theme system. Panel backgrounds use PanelBackColor, the divider color auto-syncs to BorderColor, and all child controls (search box, radio group, list box) receive theme propagation via their own ApplyTheme() methods.

radioListBox.Theme = "MaterialDark";
radioListBox.ApplyTheme(); // Entire composite updates

Examples

Basic Usage with Items

var rlb = new BeepRadioListBox
{
    Layout = RadioListBoxLayout.RadioAboveList,
    RadioStyle = RadioGroupRenderStyle.Chip,
    RadioOrientation = RadioGroupOrientation.Horizontal,
    ShowSearch = true
};

rlb.AddItem("Option A", "icons/a.svg", "Description A");
rlb.AddItem("Option B", "icons/b.svg", "Description B");
rlb.AddItem("Option C", "icons/c.svg", "Description C");

rlb.SelectionChanged += (s, e) =>
    Console.WriteLine($"Selected: {e.SelectedItem.Text} via {e.Source}");

List-Only Mode with Search

var rlb = new BeepRadioListBox
{
    Layout = RadioListBoxLayout.ListOnly,
    ListStyle = ListBoxType.Detailed,
    ShowSearch = true
};

rlb.Items = new BindingList<SimpleItem>(
    myData.Select(d => new SimpleItem { Text = d.Name, SubText = d.Info }));
rlb.SearchTextChanged += (s, e) => {
    Console.WriteLine($"Filtering for: {e.SearchText}");
};

Layout Switching at Runtime

// Switch between radio-only and full dual view
void ToggleView(BeepRadioListBox rlb)
{
    rlb.Layout = rlb.Layout == RadioListBoxLayout.RadioAboveList
        ? RadioListBoxLayout.RadioOnly
        : RadioListBoxLayout.RadioAboveList;
}
