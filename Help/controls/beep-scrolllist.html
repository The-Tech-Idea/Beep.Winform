BeepScrollList

A carousel-style scrollable list control using virtual items, supporting vertical or horizontal orientation with smooth animation, drag scrolling, and keyboard navigation

Overview

BeepScrollList is a carousel-style scrollable list control that renders items as virtual hit-testable regions without creating actual child controls. It inherits from BeepPanel and resides in TheTechIdea.Beep.Winform.Controls.Scolling. The control supports both vertical and horizontal orientation, smooth scrolling via a 60fps animation timer, mouse wheel support, drag-to-scroll, keyboard navigation (arrow keys, PageUp/Down, Home/End), hover effects, checkbox display, and optional scrollbar-style position indicators.

Key Features

  • Virtual item rendering via hit-test areas (no child controls)
  • Vertical and horizontal scroll orientation
  • Smooth animation scrolling at ~60fps with configurable speed
  • Mouse drag-to-scroll with inertia
  • Mouse wheel support
  • Scroll position indicators (mini scrollbar thumb)
  • Item selection with visual state (selected/hover/normal)
  • Keyboard navigation: Up/Down/Left/Right, PageUp/Down, Home/End
  • Optional checkboxes per item
  • Image support per item via BeepImage
  • Auto-scroll to selected item
  • Centering offset for items when total size is smaller than visible area

Theming

ApplyTheme() propagates the theme to internal drawing helpers: _button gets theme-based TextFont, BackColor, ForeColor, HoverBackColor, HoverForeColor, SelectedBackColor, SelectedForeColor from theme properties (ListUnSelectedFont, ListBackColor, ListItemForeColor, ListItemHoverBackColor, ListItemHoverForeColor, ListItemSelectedBackColor, ListItemSelectedForeColor). _image gets ApplyThemeOnImage from _currentTheme.ApplyThemeToIcons. _label gets theme font and color settings. The control itself uses SelectedBackColor, SelectedForeColor, HoverBackColor, and HoverForeColor from BeepPanel base for item state rendering.

Examples

// Vertical scroll list with images and checkboxes
var scrollList = new BeepScrollList
{
    Size = new Size(250, 400),
    Orientation = ScrollOrientation.VerticalScroll,
    ItemHeight = 40,
    ItemWidth = 230,
    ItemSpacing = 4,
    ShowCheckBox = true,
    EnableKeyboardNavigation = true,
    EnableItemHoverEffects = true
};

scrollList.ListItems.Add(new SimpleItem
{
    Text = "Dashboard",
    Value = "dashboard",
    ImagePath = Svgs.Home
});
scrollList.ListItems.Add(new SimpleItem
{
    Text = "Analytics",
    Value = "analytics",
    ImagePath = Svgs.Chart
});
scrollList.ListItems.Add(new SimpleItem
{
    Text = "Settings",
    Value = "settings",
    ImagePath = Svgs.Gear
});

scrollList.ItemSelected += (s, item) =>
{
    Console.WriteLine($"Selected: {item.Text}");
};

// Scroll to a specific item
scrollList.ScrollToItem(5);

// Horizontal carousel-style list
var horizontalList = new BeepScrollList
{
    Size = new Size(600, 120),
    Orientation = ScrollOrientation.HorizontalScroll,
    ItemHeight = 100,
    ItemWidth = 100,
    ItemSpacing = 10,
    EnableItemHoverEffects = true
};

this.Controls.Add(scrollList);

Architecture

BeepScrollList extends BeepPanel and renders items entirely through custom drawing in DrawContent without creating actual child controls. It uses hit-test areas (AddHitArea from the base HitArea system) to detect mouse interactions on virtual items. A single reusable BeepButton, BeepImage, and BeepLabel instance serve as drawing helpers (never added to Controls). A 60fps Timer drives smooth scroll animation by lerping _scrollOffset toward _targetScrollOffset. Dragging is handled through OnMouseDown/OnMouseMove/OnMouseUp state management. The control listens to ListChanged on the BindingList<SimpleItem> to auto-update virtual items. Scroll indicator thumb is drawn as a proportional rounded rectangle. ScrollToItem calculates the target position to center the item when possible, clamping to prevent overscroll.
