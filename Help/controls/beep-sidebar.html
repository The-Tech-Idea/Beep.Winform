Overview

BeepSideBar is a modern sidebar navigation control built on the painter architecture. It displays a vertical list of SimpleItem objects with icons and text, supports collapsing/expanding, and provides accordion-style expansion for items with child items. The control automatically adjusts item heights to prevent text clipping and supports theme-aware rendering with custom accent colors.

The sidebar is organized with a painter-based rendering pipeline and partial class files for animations, hit testing, and design-mode menu item editing.

Key Features

  • Collapsible sidebar — Toggle between expanded and collapsed widths with smooth animation transitions
  • Accordion expansion — Items with child SimpleItem collections display expand/collapse chevrons for hierarchical menus
  • Customizable expansion behavior — Control click-to-expand logic via ClickTogglesExpansion and ClickTogglesExpansionMode
  • Theme-aware rendering — Automatically adapts to active Beep theme with accent highlights, hover colors, and selection indicators
  • Custom accent color — Overrideable accent color for selection highlight and expand/collapse icons when UseThemeColors is disabled
  • Auto-sizing items — ItemHeight is recalculated automatically based on text and icon content to prevent clipping
  • Playelist item support — Built-in SVG icon rendering with pre-warmed icon cache for responsive first paint
  • Collapsed-state icons — SVG icon paths for expand and collapse states that swap based on item expansion
  • Shadow effects — Optional rail shadow via EnableRailShadow for modern depth styling
  • Selection animation — Animated highlight indicator that transitions between items on click
  • Hover animations — Subtle hover effects over expand/collapse icons

Theming

BeepSideBar integrates with the Beep theme system. When UseThemeColors is true (default), the sidebar draws using the active theme's palette tokens for background, foreground, hover, and selection colors. When disabled, the custom AccentColor drives selection highlights and icon colors.

Icons are processed through the StyledImagePainter pipeline, which recolors SVG icons to match the current accent or theme color. Common icons (hamburger menu, expand, collapse) are pre-rendered at multiple sizes on handle creation for responsive first paint.

BeepThemesManager.ApplyTheme("MaterialDark"); // Sidebar updates automatically
sidebar.ApplyTheme(); // Force manual refresh if needed

Examples

Basic Sidebar with Collapse Toggle

var sidebar = new BeepSideBar
{
    ExpandedWidth = 220,
    CollapsedWidth = 54,
    Dock = DockStyle.Left
};

sidebar.Items.Add(new SimpleItem { Text = "Home", ImagePath = Svgs.Home });
sidebar.Items.Add(new SimpleItem { Text = "Settings", ImagePath = Svgs.Settings });

cancelToggleButton.Click += (s, e) => sidebar.Toggle();

Accordion Menu with Child Items

var reports = new SimpleItem
{
    Text = "Reports",
    ImagePath = Svgs.Chart,
    Children = new List<SimpleItem>
    {
        new SimpleItem { Text = "Sales", ImagePath = Svgs.Dollar },
        new SimpleItem { Text = "Inventory", ImagePath = Svgs.Box }
    }
};
sidebar.Items.Add(reports);

sidebar.ItemExpansionChanged += (s, e) =>
{
    Console.WriteLine($"{e.Item.Text} expanded = {e.IsExpanded}");
};

Handling Item Clicks

sidebar.ItemClicked += item =>
{
    switch (item.Text)
    {
        case "Home": NavigateTo(new HomePage()); break;
        case "Settings": NavigateTo(new SettingsPage()); break;
    }
};
