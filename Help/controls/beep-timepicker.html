A modern time picker control with multiple format styles, validation, business-hour rules, and built-in popup selector

Overview

BeepTimePicker extends BaseControl to provide a fully themed time input. It composites an internal TextBox, a dropdown clock button, and optional spin up/down buttons. A popup panel with scrollable hours, minutes, and AM/PM lists enables quick visual selection.

Namespace: TheTechIdea.Beep.Winform.Controls  |  Assembly: TheTechIdea.Beep.Winform.Controls.dll
Inheritance: Component → Control → BaseControl → BeepTimePicker

Key Features

  • Multiple time formats — 12-hour, 24-hour, with/without seconds, hours-only, and custom format strings
  • Dropdown popup — Clock icon button opens a BeepPopupForm with hours, minutes, and AM/PM list boxes
  • Spin buttons — Optional increment/decrement buttons that respect MinuteInterval
  • Validation pipeline — Min/max time range, business hours enforcement, and customizable error display
  • Business presets — ConfigureForMeetingTime(), ConfigureForAppointment(), ConfigureForShiftTime(), ConfigureForDeadline()
  • Minute snapping — SnapToInterval rounds time to the configured MinuteInterval
  • Auto-submit — When enabled, raises TimeSubmitted immediately on selection
  • Locale-aware — Culture property controls parsing and formatting per culture
  • Validation error visuals — Error background/border colors sourced from theme tokens; tooltip error messages

Business Configuration

Four preset methods configure the picker for common business scenarios:

// Meeting scheduler: 12h, 15-min snaps, business hours required
picker.ConfigureForMeetingTime();

// Appointment booking: 12h, 30-min snaps, 8AM–6PM, auto-submit
picker.ConfigureForAppointment();

// Shift scheduling: 24h, 15-min snaps, non-empty required
picker.ConfigureForShiftTime();

// Deadline entry: 12h, auto-submit on change
picker.ConfigureForDeadline();

After calling a preset, you can further customize individual properties like BusinessStartTime or MinTime.

Theming

The control reads theme tokens for text box background/foreground, border color, and error state colors (TextBoxErrorBackColor, TextBoxErrorBorderColor). Child buttons and the popup panel inherit the theme automatically via ApplyTheme() propagation.

picker.Theme = "MaterialDark";
picker.ApplyTheme(); // Propagates to text box, buttons, and popup panel

Examples

Basic 12-Hour Picker

var picker = new BeepTimePicker
{
    TimeFormatStyle = TimeFormatStyle.TwelveHour,
    ShowDropDown = true,
    ShowSpinButtons = true,
    MinuteInterval = 5,
    SnapToInterval = true
};
picker.SelectedTime = new TimeSpan(14, 30, 0); // 2:30 PM

With Validation and Events

var picker = new BeepTimePicker
{
    MinTime = new TimeSpan(9, 0, 0),  // 9 AM
    MaxTime = new TimeSpan(17, 0, 0), // 5 PM
    AutoSubmit = true
};
picker.TimeValidationFailed += (s, e) =>
    MessageBox.Show($"Invalid: {e.ErrorMessage}");
picker.TimeSubmitted += (s, e) =>
    SaveTime(e.SelectedTime, e.Context);

Data-Binding

var picker = new BeepTimePicker();
picker.SetValue(new DateTime(2025, 6, 1, 10, 30, 0)); // from DateTime
picker.SetValue("14:45");                              // from string
var result = picker.GetValue() as TimeSpan?;           // nullable TimeSpan
