Overview

Namespace: TheTechIdea.Beep.Winform.Controls.Design.Server.Designers
Inheritance: BaseBeepControlDesigner

BeepAccordionMenuDesigner provides a rich design-time experience for the accordion sidebar menu control. It features adornment rendering — drawing empty-state hints ("Click 'Add Sample Items' to get started...") and preview-footers directly on the design surface — plus 6 designer verbs, 10 smart-tag properties, and 6 one-click style presets.

Adornment Rendering

Two visual hints drawn directly on the design surface:

  • Empty State — Dashed rectangle with text "Click 'Add Sample Items' to get started..." when the accordion has no items. This prevents a blank control from appearing confusing at design time.
  • Preview Footer — Small footer showing item count and current style name when items exist.

Examples

// At design time, use the smart-tag:
// Smart-tag > Style Presets > "Apply Material 3 Style"
// Smart-tag > Items > "Add Sample Items"
// Smart-tag > Behavior > AllowItemDragDrop = true

// Equivalent at runtime:
var accordion = new BeepAccordionMenu();
accordion.AccordionStyle = AccordionStyle.Material3;
accordion.ExpandedWidth = 240;
accordion.CollapsedWidth = 56;
accordion.ItemHeight = 44;
accordion.AnimationStep = 150;
accordion.Items.Add(new AccordionItem("Dashboard", Icons.Dashboard,
    new List<AccordionItem> {
        new AccordionItem("Analytics", Icons.Chart),
        new AccordionItem("Monitoring", Icons.Monitor)
    }));
