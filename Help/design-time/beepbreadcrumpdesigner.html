Overview

Namespace: TheTechIdea.Beep.Winform.Controls.Design.Server.Designers
Inheritance: BaseBeepControlDesigner

BeepBreadcrumpDesigner provides the design-time experience for BeepBreadcrump — a breadcrumb navigation control. The smart-tag exposes 3 properties for separator text, home icon visibility, and visual style, plus 5 style presets and 4 separator presets for quick visual configuration. No designer verbs beyond the base class.

Examples

// At design time, use smart-tag:
// Smart-tag > Style Presets > "Set Chevron Style"
// Smart-tag > Separator Presets > "Use Arrow Separator"
// Smart-tag > Appearance > ShowHomeIcon = true

// Equivalent at runtime:
var bread = new BeepBreadcrump();
bread.BreadcrumbStyle = BreadcrumbStyle.Chevron;
bread.SeparatorText = "→";
bread.ShowHomeIcon = true;
bread.Items.Add(new BreadcrumbItem("Home", () => NavigateHome()));
bread.Items.Add(new BreadcrumbItem("Products", () => NavigateProducts()));
bread.Items.Add(new BreadcrumbItem("Details", null)); // Current (non-clickable)
