Overview

Namespace: TheTechIdea.Beep.Winform.Controls.Design.Server.Designers
Inheritance: BaseBeepControlDesigner

BeepComboBoxDesigner provides the design-time experience for BeepComboBox. The smart-tag exposes 13 properties across Appearance and Behavior categories, plus 5 one-click configuration presets that reconfigure the combo box for common use cases (searchable picker, chip multi-select, command menu, simple dropdown, minimal picker). Inherits CommonBeepControlActionList for style/theme/schema.

Examples

// At design time, use smart-tag presets:
// Smart-tag > Presets > "Configure as Searchable Picker"
// -> ComboBoxType=OutlineSearchable, ShowSearchInDropdown=true, AutoComplete=true

// At runtime, equivalent API:
var combo = new BeepComboBox();
combo.ComboBoxType = ComboBoxType.OutlineSearchable;
combo.ShowSearchInDropdown = true;
combo.AutoComplete = true;
combo.AutoCompleteMode = BeepAutoCompleteMode.Suggest;
combo.AutoCompleteMinLength = 2;
combo.MaxSuggestions = 10;
combo.PlaceholderText = "Type to search...";
combo.ShowClearButton = true;
