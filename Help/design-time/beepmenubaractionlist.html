Overview

Namespace: TheTechIdea.Beep.Winform.Controls.Design.Server.Designers
Inheritance: DesignerActionList | Visibility: public sealed

BeepMenuBarActionList provides the connected smart-tag for BeepMenuBar. It directly couples to BeepMenuBarDesigner and uses its transaction-wrapped mutation methods (ExecuteAction, SetPropertyWithTransaction) for all state changes. With 5 properties and 8 action methods across 4 organized sections, it provides a compact but complete design-time configuration surface.
