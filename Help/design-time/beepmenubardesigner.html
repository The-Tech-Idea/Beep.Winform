Overview

Namespace: TheTechIdea.Beep.Winform.Controls.Design.Server.Designers
Inheritance: BaseBeepControlDesigner | Visibility: public sealed

BeepMenuBarDesigner provides the design-time experience for BeepMenuBar. It wraps all property mutations in DesignerTransaction with IComponentChangeService notifications for clean undo support and .designer.cs serialization. The smart-tag (BeepMenuBarActionList) exposes 5 properties, item manipulation actions, style presets, and height presets.

Transaction-Wrapped Mutations

internal void SetPropertyWithTransaction(string name, object? value, string desc)
{
    var prop = TypeDescriptor.GetProperties(MenuBar)[name];
    var old = prop.GetValue(MenuBar);
    if (Equals(old, value)) return;

    using (var tx = DesignerHost.CreateTransaction(desc))
    {
        ChangeService.OnComponentChanging(MenuBar, prop);
        prop.SetValue(MenuBar, value);
        ChangeService.OnComponentChanged(MenuBar, prop, old, value);
        tx.Commit();
    }
    RefreshDesignerActionUI();
}

Examples

// At design time, use smart-tag:
// Smart-tag > Items > "Load Sample Items"
// -> Creates File/Edit/Help menu structure with icons and shortcuts

// Smart-tag > Layout > "Compact (32 px)"
// -> Sets Height=32 for a compact toolbar appearance

// Equivalent at runtime:
var menu = new BeepMenuBar();
menu.ControlStyle = BeepControlStyle.Material3;
menu.Height = 44;
menu.ShowImage = true;
menu.Items.Add(new BeepMenuItem("File", null, new List<BeepMenuItem>
{
    new BeepMenuItem("New", Icons.NewFile, () => NewFile()),
    new BeepMenuItem("Open", Icons.OpenFolder, () => OpenFile()),
    new BeepMenuItem("Save", Icons.Save, () => SaveFile()),
    new BeepMenuSeparator(),
    new BeepMenuItem("Exit", null, () => Application.Exit())
}));
