ContainerControlActionList

Smart-tag for container controls -- Arrange Children and Clear All Children actions for design-time child management

Table of Contents

  • Overview

  • Examples

Overview

Namespace: TheTechIdea.Beep.Winform.Controls.Design.Server.ActionLists
Inheritance: DesignerActionList | Visibility: public

ContainerControlActionList provides a minimal smart-tag for parent/container controls via the IBeepDesignerActionHost interface. It has no properties -- only two action methods for arranging and clearing child controls at design time.

Container Actions

ActionMethodDescription
Arrange ChildrenArrangeChildren()Calls PerformLayout() on the BaseControl-derived component to arrange all child controls according to their layout rules.
Clear All ChildrenClearAllChildren()Clears the entire Controls collection of the component, removing all child controls.

Smart-Tag Sections

SectionItems
Container ActionsArrangeChildren ("Arrange Children"), ClearAllChildren ("Clear All Children")

Examples

// At design time, container controls (Panels, GroupBoxes) show:
// Smart-tag > Container Actions > "Arrange Children"
//   -> Calls PerformLayout() to snap children into position

// Smart-tag > Container Actions > "Clear All Children"
//   -> Removes all child controls from the container

// Implementation delegates to BaseControl:
var control = (BaseControl)_designer.Component;
control.PerformLayout();      // ArrangeChildren
control.Controls.Clear();     // ClearAllChildren
