BeepDashboardWidget

Composite dashboard widget that arranges multiple KPIs, charts, and timelines in a single panel with six layout styles.

Stable v1.0.164 Beep Widgets / Dashboard

Overview

BeepDashboardWidget is a paint-based WinForms widget that inherits from BaseControl. It uses an IWidgetPainter pipeline to render rich visuals without child controls. The Style property selects one of 6 built-in render modes; changing it at runtime immediately swaps the painter and triggers a repaint.

Key Features

🗂️ Six Layouts

Grid, timeline, comparison, and analytics — all from one widget.

📊 DashboardMetric

Simple DashboardMetric model (Title, Value, Trend, Color) populates all styles.

🎨 Gradient Panels

Built-in LinearGradient support for the panel background.

🖱️ Card Click Events

MetricClicked fires with the specific DashboardMetric the user clicked.

📐 Configurable Grid

Columns and Rows properties control the metric/chart grid density.

🔄 Live Update

Update Metrics collection and call Invalidate() to refresh in place.

Basic Usage

var dash = new BeepDashboardWidget
{
    Style   = DashboardWidgetStyle.MultiMetric,
    Title   = "Q4 KPIs",
    Columns = 2,
    Metrics = new List
    {
        new DashboardMetric { Title = "Revenue", Value = "$127K",  Trend = "+12%", Color = Color.Green  },
        new DashboardMetric { Title = "Users",   Value = "23,456", Trend = "+8%",  Color = Color.Blue   },
        new DashboardMetric { Title = "Orders",  Value = "1,234",  Trend = "-2%",  Color = Color.Red    },
        new DashboardMetric { Title = "Growth",  Value = "15.7%",  Trend = "+5%",  Color = Color.Orange },
    },
    Size = new Size(420, 260)
};
this.Controls.Add(dash);

Event Handling

dash.MetricClicked += (s, e) =>
{
    var m = (DashboardMetric)e.Data;
    DrillDown(m.Title);
};

Best Practices

  • Set Style before adding the widget to the form to avoid double repaints.
  • Adjust Size to fit your layout — defaults are minimum comfortable sizes.
  • Read BeepEventDataArgs.Data in event handlers for context-specific click data.
  • Call Invalidate() after updating data properties to trigger a repaint.
  • Let theme colors flow from BeepStyling.CurrentTheme — avoid overriding color properties unless customization is explicitly desired.
