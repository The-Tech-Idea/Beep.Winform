?? Overview

BeepDialogBox is a flexible dialog control that extends BeepPanel to provide customizable message dialogs, confirmations, warnings, and user prompts. It offers standard dialog functionality like OK/Cancel buttons, but with enhanced styling, theming, and customization options.

As a descendent of the BeepPanel class, BeepDialogBox inherits all panel features such as rounded corners, shadows, and theming capabilities, while adding dialog-specific functionality like multiple button options, dialog results, and built-in close button.

?? Multiple Button Support

Primary, secondary, and optional third buttons with customizable text and colors

?? Dialog Results

Standard dialog result values returned based on user interaction

Configurable close button in the title bar for quick dismissal

Optional icon display for visual context (information, warning, error, etc.)

Full support for Beep theming system with dialog-specific styling

Automatically arranges buttons and content based on the dialog size

?? Basic Usage

Simple Confirmation Dialog

// Create a basic confirmation dialog
var confirmDialog = new BeepDialogBox
{
    TitleText = "Confirm Action",
    PrimaryButtonText = "OK",
    SecondaryButtonText = "Cancel",
    Size = new Size(400, 200)
};

// Add content to the dialog (use the content area of the BeepPanel)
BeepLabel messageLabel = new BeepLabel
{
    Text = "Are you sure you want to proceed with this action?",
    AutoSize = true,
    Location = new Point(20, 50)
};
confirmDialog.Controls.Add(messageLabel);

// Handle button clicks
confirmDialog.PrimaryButtonClicked += (sender, e) => 
{
    confirmDialog.DialogResult = BeepDialogResult.OK;
    confirmDialog.Hide();
};

confirmDialog.SecondaryButtonClicked += (sender, e) => 
{
    confirmDialog.DialogResult = BeepDialogResult.Cancel;
    confirmDialog.Hide();
};

// Show dialog
confirmDialog.ShowDialog();
Tip: Dialog Results

The BeepDialogBox uses the BeepDialogResult enum which provides standard dialog result values like OK, Cancel, Yes, No, etc. You can check the dialog result after it closes to determine the user's action.

?? Properties

Button Properties

Property Type Default Description
PrimaryButtonText string "OK" Text displayed on the primary button
SecondaryButtonText string "Cancel" Text displayed on the secondary button
PrimaryButtonColor Color Theme-defined Background color of the primary button
SecondaryButtonColor Color Theme-defined Background color of the secondary button
DialogResult BeepDialogResult None Result value indicating which button was clicked

Inherited Panel Properties

As a subclass of BeepPanel, BeepDialogBox inherits all the standard BeepPanel properties including:

TitleText string

Text displayed in the dialog's title bar

ShowTitle bool

Whether to display the title bar

ShowShadow bool

Whether to display a shadow effect

BorderRadius int

Radius for rounded corners of the dialog

?? Events

Event Description
PrimaryButtonClicked Triggered when the primary button is clicked
SecondaryButtonClicked Triggered when the secondary button is clicked
ThirdButtonClicked Triggered when the third button is clicked (if visible)
CloseButtonClicked Triggered when the close button in the title bar is clicked

Event Handling Example

// Create a dialog
var dialog = new BeepDialogBox
{
    TitleText = "File Save Options",
    PrimaryButtonText = "Save",
    SecondaryButtonText = "Don't Save",
};

// Set up a third button
dialog._thirdButton.Text = "Cancel";
dialog._thirdButton.Visible = true;

// Add event handlers
dialog.PrimaryButtonClicked += (sender, e) => {
    // Save file
    SaveCurrentDocument();
    dialog.DialogResult = BeepDialogResult.Yes;
    dialog.Hide();
};

dialog.SecondaryButtonClicked += (sender, e) => {
    // Discard changes
    dialog.DialogResult = BeepDialogResult.No;
    dialog.Hide();
};

dialog.ThirdButtonClicked += (sender, e) => {
    // Cancel operation
    dialog.DialogResult = BeepDialogResult.Cancel;
    dialog.Hide();
};

dialog.CloseButtonClicked += (sender, e) => {
    // Same as cancel
    dialog.DialogResult = BeepDialogResult.Cancel;
};

Information Dialog

/// 
/// Shows an information dialog with an OK button
/// 
public BeepDialogResult ShowInformation(string title, string message)
{
    // Create the dialog
    var dialog = new BeepDialogBox
    {
        TitleText = title,
        PrimaryButtonText = "OK",
        Size = new Size(400, 200),
        StartPosition = FormStartPosition.CenterParent,
    };
    
    // Hide secondary button
    dialog._secondaryButton.Visible = false;
    
    // Create message label
    BeepLabel messageLabel = new BeepLabel
    {
        Text = message,
        AutoSize = true,
        Location = new Point(20, 60),
        MaximumSize = new Size(360, 0)
    };
    dialog.Controls.Add(messageLabel);
    
    // Add information icon
    BeepImage iconImage = new BeepImage
    {
        ImagePath = "icons/information.svg",
        Size = new Size(32, 32),
        Location = new Point(20, 20)
    };
    dialog.Controls.Add(iconImage);
    
    // Handle button click
    dialog.PrimaryButtonClicked += (sender, e) =>
    {
        dialog.DialogResult = BeepDialogResult.OK;
        dialog.Hide();
    };
    
    // Show dialog and return result
    dialog.ShowDialog();
    return dialog.DialogResult;
}

Custom User Input Dialog

/// 
/// Shows a text input dialog and returns the entered text
/// 
public string ShowInputDialog(string title, string prompt, string defaultValue = "")
{
    string result = defaultValue;
    
    // Create the dialog
    var dialog = new BeepDialogBox
    {
        TitleText = title,
        PrimaryButtonText = "OK",
        SecondaryButtonText = "Cancel",
        Size = new Size(450, 200),
        StartPosition = FormStartPosition.CenterParent
    };
    
    // Create prompt label
    BeepLabel promptLabel = new BeepLabel
    {
        Text = prompt,
        AutoSize = true,
        Location = new Point(20, 20)
    };
    
    // Create text input
    BeepTextBox inputBox = new BeepTextBox
    {
        Text = defaultValue,
        Location = new Point(20, 50),
        Width = 400,
        Height = 30
    };
    
    // Add controls to dialog
    dialog.Controls.Add(promptLabel);
    dialog.Controls.Add(inputBox);
    
    // Handle button clicks
    dialog.PrimaryButtonClicked += (sender, e) =>
    {
        result = inputBox.Text;
        dialog.DialogResult = BeepDialogResult.OK;
        dialog.Hide();
    };
    
    dialog.SecondaryButtonClicked += (sender, e) =>
    {
        dialog.DialogResult = BeepDialogResult.Cancel;
        dialog.Hide();
    };
    
    // Show dialog
    var dialogResult = dialog.ShowDialog();
    
    // Return input if OK was clicked, otherwise return empty string
    return dialogResult == BeepDialogResult.OK ? result : string.Empty;
}

Creating a Dialog Helper Class

/// 
/// Helper class for commonly used dialogs
/// 
public class BeepDialogs
{
    // Show information message
    public static BeepDialogResult Information(string message, string title = "Information")
    {
        return ShowDialog(title, message, "OK", null, null, "icons/information.svg");
    }
    
    // Show warning message
    public static BeepDialogResult Warning(string message, string title = "Warning")
    {
        return ShowDialog(title, message, "OK", null, null, "icons/warning.svg");
    }
    
    // Show error message
    public static BeepDialogResult Error(string message, string title = "Error")
    {
        return ShowDialog(title, message, "OK", null, null, "icons/error.svg");
    }
    
    // Show confirmation dialog
    public static BeepDialogResult Confirm(string message, string title = "Confirm")
    {
        return ShowDialog(title, message, "Yes", "No", null, "icons/question.svg");
    }
    
    // Show Yes/No/Cancel dialog
    public static BeepDialogResult YesNoCancel(string message, string title = "Question")
    {
        return ShowDialog(title, message, "Yes", "No", "Cancel", "icons/question.svg");
    }
    
    // Core dialog implementation
    private static BeepDialogResult ShowDialog(string title, string message, 
        string primaryText, string secondaryText, string thirdText, string iconPath)
    {
        var dialog = new BeepDialogBox
        {
            TitleText = title,
            PrimaryButtonText = primaryText,
            Size = new Size(450, 220),
            StartPosition = FormStartPosition.CenterScreen
        };
        
        // Configure buttons
        if (string.IsNullOrEmpty(secondaryText))
        {
            dialog._secondaryButton.Visible = false;
        }
        else
        {
            dialog.SecondaryButtonText = secondaryText;
        }
        
        if (!string.IsNullOrEmpty(thirdText))
        {
            dialog._thirdButton.Text = thirdText;
            dialog._thirdButton.Visible = true;
        }
        
        // Add icon if provided
        if (!string.IsNullOrEmpty(iconPath))
        {
            BeepImage iconImage = new BeepImage
            {
                ImagePath = iconPath,
                Size = new Size(48, 48),
                Location = new Point(20, 20)
            };
            dialog.Controls.Add(iconImage);
        }
        
        // Create message label with proper positioning
        int messageX = !string.IsNullOrEmpty(iconPath) ? 80 : 20;
        BeepLabel messageLabel = new BeepLabel
        {
            Text = message,
            AutoSize = true,
            Location = new Point(messageX, 30),
            MaximumSize = new Size(350, 0)
        };
        dialog.Controls.Add(messageLabel);
        
        // Set up result handling
        dialog.PrimaryButtonClicked += (s, e) => {
            dialog.DialogResult = primaryText == "Yes" ? 
                BeepDialogResult.Yes : BeepDialogResult.OK;
            dialog.Hide();
        };
        
        if (!string.IsNullOrEmpty(secondaryText))
        {
            dialog.SecondaryButtonClicked += (s, e) => {
                dialog.DialogResult = secondaryText == "No" ? 
                    BeepDialogResult.No : BeepDialogResult.Cancel;
                dialog.Hide();
            };
        }
        
        if (!string.IsNullOrEmpty(thirdText))
        {
            dialog.ThirdButtonClicked += (s, e) => {
                dialog.DialogResult = BeepDialogResult.Cancel;
                dialog.Hide();
            };
        }
        
        // Show dialog and return result
        dialog.ShowDialog();
        return dialog.DialogResult;
    }
}

Using the Dialog Helper Class

// Show information dialog
BeepDialogs.Information("Operation completed successfully.");

// Show warning dialog
BeepDialogs.Warning("Low disk space available.");

// Show error dialog
BeepDialogs.Error("An error occurred while saving the file.");

// Show confirmation dialog and check result
if (BeepDialogs.Confirm("Are you sure you want to delete this item?") == BeepDialogResult.Yes)
{
    DeleteItem();
}

// Show Yes/No/Cancel dialog
BeepDialogResult result = BeepDialogs.YesNoCancel("Save changes before closing?");
switch (result)
{
    case BeepDialogResult.Yes:
        SaveChanges();
        CloseApplication();
        break;
    case BeepDialogResult.No:
        CloseApplication();
        break;
    case BeepDialogResult.Cancel:
        // Do nothing, continue editing
        break;
}

?? Theming

BeepDialogBox fully integrates with the Beep theming system, inheriting all panel theming capabilities while adding dialog-specific theming features.

Theme Integration

// Apply a theme to the dialog
dialog.Theme = "DarkTheme";
dialog.ApplyTheme();

// Theme properties that affect BeepDialogBox:
// - PanelBackColor: Dialog background color
// - PrimaryTextColor: Primary button text color
// - SecondaryTextColor: Secondary button text color
// - ButtonBackColor: Default button background color
// - DialogCloseButtonForeColor: Close button color
Tip: Custom Button Styling

You can access the button controls directly to apply custom styling beyond what the theme provides:

// Apply custom styling to the primary button
dialog._primaryButton.UIVariant = ReactUIVariant.Contained;
dialog._primaryButton.UIColor = ReactUIColor.Success;

?? Best Practices

Dialog Content

  • Clear messages: Keep dialog messages concise and easy to understand
  • Descriptive titles: Use specific titles that indicate the dialog's purpose
  • Appropriate icons: Use icons that match the dialog type (information, warning, error)
  • Button clarity: Make button labels clear and action-oriented

Dialog Behavior

  • Modal display: Use ShowDialog() for important prompts that require user action
  • Close handlers: Always handle the close button event to set an appropriate dialog result
  • Keyboard support: Ensure Enter and Escape keys work as expected
  • Focus management: Set initial focus to the most likely action or input field

Layout Considerations

  • Appropriate sizing: Size dialogs appropriately for their content
  • Center positioning: Center dialogs relative to their parent window
  • Content spacing: Use consistent padding and spacing for dialog content
Watch Out!

Avoid overusing dialogs for non-critical information that could be presented in-line or with less disruptive notifications. Dialog fatigue can lead to users dismissing important messages without reading them.
