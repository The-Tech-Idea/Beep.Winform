# BorderlessMinimal Layout Plan

Intent
- Very light, borderless look (like clean dashboards). No grid lines, no stripes, extra whitespace.

Grid Tweaks
- RowHeight: 34
- ColumnHeaderHeight: 42
- ShowGridLines: false
- ShowRowStripes: false
- UseHeaderGradient: false
- UseHeaderHoverEffects: false
- UseBoldHeaderText: false
- HeaderCellPadding: 8

Alignment Heuristics
- First column: Left
- Rest: Left (or Center if numeric/percent — left in first pass).
