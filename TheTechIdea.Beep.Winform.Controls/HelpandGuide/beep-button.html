?? BeepButton Control

BeepButton is an enhanced button control that extends BeepControl with advanced features including image support, animations, popup menus, Material Design ripple effects, long press gestures, and badge notifications.

?? Inheritance: BeepButton inherits from BeepControl, providing all base functionality plus button-specific features.

? Key Features

???

Image Support

Full SVG and raster image support with theme-aware coloring and flexible positioning.

??

Popup Lists

Built-in dropdown menu functionality with cascading submenus and item selection.

??

Ripple Effects

Material Design splash animations and ripple effects on click interactions.

??

Long Press

Long press gesture detection for context menus and secondary actions.

???

Badge Support

Notification badges with customizable shapes, colors, and positioning.

??

Auto Sizing

Automatic sizing based on content with scaled font support.

?? Basic Usage

Simple Button

var button = new BeepButton
{
    Text = "Click Me",
    Size = new Size(120, 40),
    Location = new Point(10, 10),
    Theme = "DefaultTheme"
};

button.Click += (sender, e) => {
    MessageBox.Show("Button clicked!");
};

Button with Image

var imageButton = new BeepButton
{
    Text = "Save Document",
    ImagePath = "icons/save.svg",
    TextImageRelation = TextImageRelation.ImageBeforeText,
    MaxImageSize = new Size(24, 24),
    Size = new Size(150, 45),
    ApplyThemeOnImage = true
};

?? Properties Reference

Image Properties

Property Type Default Description
ImagePath string null Path to image file (SVG, PNG, JPG, etc.)
Image Image null Direct image object assignment
MaxImageSize Size 32x32 Maximum size for the image
TextImageRelation TextImageRelation ImageBeforeText Positioning of text relative to image
ImageAlign ContentAlignment MiddleLeft Image alignment within the button
ApplyThemeOnImage bool false Apply theme colors to SVG images

Text Properties

Property Type Default Description
Text string "" Button text content
TextAlign ContentAlignment MiddleCenter Text alignment within the button
TextFont Font Arial, 10pt Font used for button text
HideText bool false Hide text display (image only button)
UseScaledFont bool false Automatically scale font to fit button size

Popup List Properties

Property Type Default Description
PopupMode bool false Enable popup dropdown functionality
ListItems BindingList<SimpleItem> empty Items to display in popup list
SelectedItem SimpleItem null Currently selected item
SelectedIndex int -1 Index of selected item
PopPosition BeepPopupFormPosition Bottom Position of popup relative to button
IsPopupOpen bool false Whether popup is currently open

Animation Properties

Property Type Default Description
SplashColor Color Gray Color of the splash/ripple effect
EnableRippleEffect bool false Enable Material Design ripple effect

Behavior Properties

Property Type Default Description
AutoSize bool false Automatically size button based on content
IsStillButton bool false Maintain hover state appearance
IsColorFromTheme bool true Use colors from current theme

?? Events

Event Args Description
Click EventArgs Standard button click event
LongPress EventArgs Fired when button is long-pressed
DoubleClickAction EventArgs Fired on double-click
ImageClicked BeepEventDataArgs Fired when image area is clicked
SelectedItemChanged SelectedItemChangedEventArgs Fired when popup list selection changes

Popup Menu Button

var menuButton = new BeepButton
{
    Text = "Actions ?",
    PopupMode = true,
    PopPosition = BeepPopupFormPosition.Bottom,
    UIVariant = ReactUIVariant.Outlined
};

// Add menu items
menuButton.ListItems.Add(new SimpleItem { Text = "Edit", Value = "edit", ImagePath = "icons/edit.svg" });
menuButton.ListItems.Add(new SimpleItem { Text = "Delete", Value = "delete", ImagePath = "icons/delete.svg" });
menuButton.ListItems.Add(new SimpleItem { Text = "Copy", Value = "copy", ImagePath = "icons/copy.svg" });

// Handle selection
menuButton.SelectedItemChanged += (s, e) => {
    switch (e.SelectedItem.Value.ToString())
    {
        case "edit":
            EditItem();
            break;
        case "delete":
            DeleteItem();
            break;
        case "copy":
            CopyItem();
            break;
    }
};

    {
        case "save": Save(); break;
        case "saveas": SaveAs(); break;
        case "saveall": SaveAll(); break;
    }

State Management

public class StatefulButton : BeepButton
{
    public enum ButtonState
    {
        Normal,
        Loading,
        Success,
        Error
    }

    private ButtonState _state = ButtonState.Normal;
    
    public ButtonState State
    {
        get => _state;
        set
        {
            _state = value;
            UpdateButtonAppearance();
        }
    }

    private void UpdateButtonAppearance()
    {
        switch (State)
        {
            case ButtonState.Normal:
                Text = "Submit";
                UIColor = ReactUIColor.Primary;
                Enabled = true;
                break;
                
            case ButtonState.Loading:
                Text = "Loading...";
                UIColor = ReactUIColor.Default;
                Enabled = false;
                break;
                
            case ButtonState.Success:
                Text = "Success!";
                UIColor = ReactUIColor.Success;
                Enabled = true;
                break;
                
            case ButtonState.Error:
                Text = "Error - Retry";
                UIColor = ReactUIColor.Error;
                Enabled = true;
                break;
        }
        
        Invalidate();
    }
}
