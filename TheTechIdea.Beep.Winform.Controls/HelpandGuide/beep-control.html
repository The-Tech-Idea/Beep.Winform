?? BeepControl Base Class

BeepControl is the foundational base class for all Beep UI components. It provides comprehensive functionality including theming, React-style properties, Material Design support, animations, hit testing, and advanced drawing capabilities.

?? Inheritance: All Beep controls inherit from BeepControl, giving them access to all base functionality described here.

?? Core Features

??

Theming System

Built-in theme support with automatic style inheritance and custom theme creation.

??

React-Style Properties

Modern component variants, sizes, colors, and elevation properties like React UI libraries.

??

Material Design

Material Design border variants, floating labels, and ripple effects.

??

Hit Testing

Advanced hit testing for complex interactions and custom drawing areas.

??

Animations

Built-in animations including fade, slide, and easing functions.

???

External Drawing

Advanced drawing system for custom child control rendering.

Theme Properties

ShowAllBorders bool false Whether to show borders on all sides

?? Usage Examples

Basic Setup

// Create a basic BeepControl-derived control
var myControl = new BeepPanel
{
    Theme = "DefaultTheme",
    IsRounded = true,
    BorderRadius = 12,
    ShowShadow = true,
    ShadowOpacity = 0.3f,
    Size = new Size(300, 200),
    UIVariant = ReactUIVariant.Outlined,
    UIColor = ReactUIColor.Primary
};

React-Style Configuration

// Configure React-style properties
var reactButton = new BeepButton
{
    Text = "React Style Button",
    UIVariant = ReactUIVariant.Contained,
    UISize = ReactUISize.Large,
    UIColor = ReactUIColor.Success,
    UIElevation = ReactUIElevation.Medium,
    UIShape = ReactUIShape.Pill,
    UIFullWidth = true
};

Material Design Setup

// Configure Material Design properties
var materialTextBox = new BeepTextBox
{
    MaterialBorderVariant = MaterialTextFieldVariant.Outlined,
    LabelText = "Email Address",
    HelperText = "Enter your email",
    FloatingLabel = true,
    EnableRippleEffect = true,
    FocusBorderColor = Color.Blue
};

Animation Example

// Animate control appearance
var animatedPanel = new BeepPanel();
animatedPanel.ShowWithAnimation(DisplayAnimationType.SlideAndFade, parentControl);

// Or with dropdown animation
animatedPanel.ShowWithDropdownAnimation(parentControl);

Hit Testing Example

// Add hit test areas
var panel = new BeepPanel();

// Add rectangular hit area
panel.AddHitArea("button1", new Rectangle(10, 10, 100, 30), null, () => {
    MessageBox.Show("Button 1 clicked!");
});

// Add hit area for child component
var childButton = new BeepButton { Text = "Child" };
panel.AddHitArea("childButton", childButton, () => {
    MessageBox.Show("Child button clicked!");
});

External Drawing Example

// Parent control with external drawing
public partial class ParentControl : BeepControl
{
    protected override void OnParentChanged(EventArgs e)
    {
        base.OnParentChanged(e);
        
        if (Parent is BeepControl beepParent)
        {
            // Register for external drawing
            beepParent.AddChildExternalDrawing(
                this,
                DrawBadgeExternally,
                DrawingLayer.AfterAll
            );
        }
    }

    private void DrawBadgeExternally(Graphics g, Rectangle childBounds)
    {
        // Custom drawing logic
        if (!string.IsNullOrEmpty(BadgeText))
        {
            var badgeRect = new Rectangle(
                childBounds.Right - 20,
                childBounds.Top - 10,
                20, 20
            );
            
            using (var brush = new SolidBrush(Color.Red))
            {
                g.FillEllipse(brush, badgeRect);
            }
            
            TextRenderer.DrawText(g, BadgeText, Font, badgeRect, Color.White,
                TextFormatFlags.HorizontalCenter | TextFormatFlags.VerticalCenter);
        }
    }
}

?? Event Handling

Key Events

Event Description
PropertyChanged Fired when a property value changes
OnSelected Fired when the control is selected
OnValidate Fired during validation
OnValueChanged Fired when the control value changes
HitDetected Fired when a hit test area is activated
TabKeyPressed Fired when Tab key is pressed
EnterKeyPressed Fired when Enter key is pressed

Event Handler Example

var control = new BeepPanel();

// Handle property changes
control.PropertyChanged += (sender, e) => {
    Console.WriteLine($"Property {e.PropertyName} changed to {e.NewValue}");
};

// Handle hit detection
control.HitDetected += (sender, e) => {
    Console.WriteLine($"Hit detected on {e.HitTest.Name}");
};

// Handle key presses
control.EnterKeyPressed += (sender, e) => {
    // Handle Enter key
    ProcessForm();
};

control.TabKeyPressed += (sender, e) => {
    // Handle Tab navigation
    MoveToNextControl();
};
?? Best Practice: Always unsubscribe from events in the Dispose method to prevent memory leaks.
