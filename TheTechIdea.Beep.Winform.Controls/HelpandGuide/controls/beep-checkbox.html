Overview

BeepCheckBox is an enhanced checkbox control that extends the standard Windows Forms CheckBox with modern styling, three-state support, improved theming, and better visual feedback for user interactions.

Built on the BeepControl foundation, BeepCheckBox provides smooth animations, customizable check marks, state-aware theming, and support for indeterminate states commonly used in tree views and option lists.

Basic Usage

Simple CheckBox

// Create a basic checkbox
var checkBox = new BeepCheckBox
{
    Text = "Enable notifications",
    Size = new Size(200, 30),
    Location = new Point(10, 10),
    Checked = false
};

// Add to form
this.Controls.Add(checkBox);

// Handle check changed event
checkBox.CheckedChanged += (sender, e) => {
    MessageBox.Show($"Notifications: {checkBox.Checked}");
};

Three-State CheckBox

// Create checkbox with three states
var threeStateCheckBox = new BeepCheckBox
{
    Text = "Select All Items",
    ThreeState = true,
    CheckState = CheckState.Indeterminate,
    Size = new Size(150, 30),
    Location = new Point(10, 50)
};

// Handle state changes
threeStateCheckBox.CheckStateChanged += (sender, e) => {
    switch (threeStateCheckBox.CheckState)
    {
        case CheckState.Checked:
            SelectAllItems();
            break;
        case CheckState.Unchecked:
            DeselectAllItems();
            break;
        case CheckState.Indeterminate:
            // Some items selected
            break;
    }
};
üí° Usage Tip

Use three-state checkboxes for "Select All" functionality where some child items might be selected while others are not.

Check States

BeepCheckBox supports three distinct states for flexible user interaction:

State Types

State Value Visual Use Case
Unchecked false Empty checkbox Option is not selected
Checked true Checkbox with checkmark Option is selected
Indeterminate null Checkbox with filled square Partial selection (some child items selected)

State Management

// Set specific states
checkBox.CheckState = CheckState.Checked;
checkBox.CheckState = CheckState.Unchecked;
checkBox.CheckState = CheckState.Indeterminate;

// Check boolean property (ignores indeterminate)
bool isChecked = checkBox.Checked;

// Check for specific state
if (checkBox.CheckState == CheckState.Indeterminate)
{
    // Handle partial selection
    UpdateChildItems();
}

// Toggle through states programmatically
checkBox.CheckState = checkBox.CheckState == CheckState.Checked 
    ? CheckState.Unchecked 
    : CheckState.Checked;

Event Handling Examples

var checkBox = new BeepCheckBox 
{ 
    Text = "Sample CheckBox",
    ThreeState = true
};

// Handle checked state changes (boolean only)
checkBox.CheckedChanged += (sender, e) => {
    Console.WriteLine($"Checked: {checkBox.Checked}");
    UpdateRelatedControls(checkBox.Checked);
};

// Handle all state changes (including indeterminate)
checkBox.CheckStateChanged += (sender, e) => {
    switch (checkBox.CheckState)
    {
        case CheckState.Checked:
            Console.WriteLine("Fully selected");
            break;
        case CheckState.Unchecked:
            Console.WriteLine("Not selected");
            break;
        case CheckState.Indeterminate:
            Console.WriteLine("Partially selected");
            break;
    }
};

// Handle click events for custom logic
checkBox.Click += (sender, e) => {
    Console.WriteLine("CheckBox clicked");
    // Perform custom validation or logging
};

Examples

Settings Panel with Multiple Options

public class SettingsPanel : UserControl
{
    private BeepCheckBox[] settingsCheckBoxes;
    
    public SettingsPanel()
    {
        InitializeSettings();
    }
    
    private void InitializeSettings()
    {
        var settings = new[]
        {
            "Enable auto-save",
            "Show notifications",
            "Dark mode",
            "Check for updates",
            "Enable sound effects",
            "Show tooltips"
        };
        
        settingsCheckBoxes = new BeepCheckBox[settings.Length];
        
        for (int i = 0; i < settings.Length; i++)
        {
            settingsCheckBoxes[i] = new BeepCheckBox
            {
                Text = settings[i],
                Size = new Size(200, 30),
                Location = new Point(20, 20 + (i * 35)),
                Checked = GetSettingValue(settings[i]) // Load from config
            };
            
            // Handle changes
            int index = i; // Capture for closure
            settingsCheckBoxes[i].CheckedChanged += (s, e) => {
                SaveSettingValue(settings[index], settingsCheckBoxes[index].Checked);
            };
            
            this.Controls.Add(settingsCheckBoxes[i]);
        }
    }
    
    public void ApplyTheme(string themeName)
    {
        foreach (var checkBox in settingsCheckBoxes)
        {
            checkBox.Theme = themeName;
            checkBox.ApplyTheme();
        }
    }
}

Tree View with Select All

public class SelectableTreePanel : UserControl
{
    private BeepCheckBox selectAllCheckBox;
    private List itemCheckBoxes;
    
    public SelectableTreePanel()
    {
        InitializeControls();
    }
    
    private void InitializeControls()
    {
        // Master "Select All" checkbox
        selectAllCheckBox = new BeepCheckBox
        {
            Text = "Select All",
            ThreeState = true,
            Size = new Size(100, 25),
            Location = new Point(10, 10),
            Font = new Font("Segoe UI", 9f, FontStyle.Bold)
        };
        selectAllCheckBox.CheckStateChanged += SelectAllCheckBox_CheckStateChanged;
        
        // Individual item checkboxes
        itemCheckBoxes = new List();
        var items = new[] { "Item 1", "Item 2", "Item 3", "Item 4", "Item 5" };
        
        for (int i = 0; i < items.Length; i++)
        {
            var itemCheckBox = new BeepCheckBox
            {
                Text = items[i],
                Size = new Size(150, 25),
                Location = new Point(30, 40 + (i * 30))
            };
            itemCheckBox.CheckedChanged += ItemCheckBox_CheckedChanged;
            
            itemCheckBoxes.Add(itemCheckBox);
            this.Controls.Add(itemCheckBox);
        }
        
        this.Controls.Add(selectAllCheckBox);
        UpdateSelectAllState();
    }
    
    private void SelectAllCheckBox_CheckStateChanged(object sender, EventArgs e)
    {
        if (selectAllCheckBox.CheckState != CheckState.Indeterminate)
        {
            bool newState = selectAllCheckBox.Checked;
            
            // Update all items without triggering events
            foreach (var item in itemCheckBoxes)
            {
                item.CheckedChanged -= ItemCheckBox_CheckedChanged;
                item.Checked = newState;
                item.CheckedChanged += ItemCheckBox_CheckedChanged;
            }
        }
    }
    
    private void ItemCheckBox_CheckedChanged(object sender, EventArgs e)
    {
        UpdateSelectAllState();
    }
    
    private void UpdateSelectAllState()
    {
        int checkedCount = itemCheckBoxes.Count(cb => cb.Checked);
        
        selectAllCheckBox.CheckStateChanged -= SelectAllCheckBox_CheckStateChanged;
        
        if (checkedCount == 0)
        {
            selectAllCheckBox.CheckState = CheckState.Unchecked;
        }
        else if (checkedCount == itemCheckBoxes.Count)
        {
            selectAllCheckBox.CheckState = CheckState.Checked;
        }
        else
        {
            selectAllCheckBox.CheckState = CheckState.Indeterminate;
        }
        
        selectAllCheckBox.CheckStateChanged += SelectAllCheckBox_CheckStateChanged;
    }
    
    public List GetSelectedItems()
    {
        return itemCheckBoxes
            .Where(cb => cb.Checked)
            .Select(cb => cb.Text)
            .ToList();
    }
}

Permission Matrix

public class PermissionMatrix : UserControl
{
    private Dictionary permissionMatrix;
    
    public PermissionMatrix()
    {
        CreatePermissionMatrix();
    }
    
    private void CreatePermissionMatrix()
    {
        var roles = new[] { "Admin", "Manager", "User", "Guest" };
        var permissions = new[] { "Read", "Write", "Delete", "Execute" };
        
        permissionMatrix = new Dictionary();
        
        // Create header labels
        for (int i = 0; i < permissions.Length; i++)
        {
            var headerLabel = new Label
            {
                Text = permissions[i],
                Size = new Size(80, 20),
                Location = new Point(100 + (i * 90), 10),
                TextAlign = ContentAlignment.MiddleCenter,
                Font = new Font("Segoe UI", 9f, FontStyle.Bold)
            };
            this.Controls.Add(headerLabel);
        }
        
        // Create role rows
        for (int roleIndex = 0; roleIndex < roles.Length; roleIndex++)
        {
            string role = roles[roleIndex];
            var roleLabel = new Label
            {
                Text = role,
                Size = new Size(80, 25),
                Location = new Point(10, 40 + (roleIndex * 35)),
                TextAlign = ContentAlignment.MiddleRight,
                Font = new Font("Segoe UI", 9f, FontStyle.Bold)
            };
            this.Controls.Add(roleLabel);
            
            // Create permission checkboxes for this role
            var rolePermissions = new BeepCheckBox[permissions.Length];
            
            for (int permIndex = 0; permIndex < permissions.Length; permIndex++)
            {
                rolePermissions[permIndex] = new BeepCheckBox
                {
                    Size = new Size(25, 25),
                    Location = new Point(130 + (permIndex * 90), 40 + (roleIndex * 35)),
                    CheckAlign = ContentAlignment.MiddleCenter,
                    Text = "",
                    Tag = new { Role = role, Permission = permissions[permIndex] }
                };
                
                rolePermissions[permIndex].CheckedChanged += PermissionCheckBox_CheckedChanged;
                this.Controls.Add(rolePermissions[permIndex]);
            }
            
            permissionMatrix[role] = rolePermissions;
        }
        
        LoadPermissions();
    }
    
    private void PermissionCheckBox_CheckedChanged(object sender, EventArgs e)
    {
        var checkBox = sender as BeepCheckBox;
        var info = (dynamic)checkBox.Tag;
        
        // Save permission change
        SavePermission(info.Role, info.Permission, checkBox.Checked);
        
        // Update dependent permissions (e.g., Write requires Read)
        UpdateDependentPermissions(info.Role, info.Permission, checkBox.Checked);
    }
    
    private void UpdateDependentPermissions(string role, string permission, bool granted)
    {
        var rolePermissions = permissionMatrix[role];
        
        if (permission == "Write" && granted)
        {
            // Write permission requires Read
            rolePermissions[0].Checked = true; // Read is index 0
        }
        else if (permission == "Delete" && granted)
        {
            // Delete permission requires Read and Write
            rolePermissions[0].Checked = true; // Read
            rolePermissions[1].Checked = true; // Write
        }
    }
}

Theming

BeepCheckBox integrates seamlessly with the Beep theming system, automatically adapting colors and styles.

Theme Application

// Apply theme to individual checkbox
checkBox.Theme = "DarkTheme";
checkBox.ApplyTheme();

// Theme properties automatically applied:
// - CheckBoxBackColor (unchecked background)
// - CheckBoxForeColor (text color)
// - CheckBoxCheckedBackColor (checked background)
// - CheckBoxCheckedForeColor (checked text color)
// - CheckBoxBorderColor (border color)
// - CheckBoxCheckMarkColor (checkmark color)

// Custom theme overrides
checkBox.CheckedBackColor = Color.LightBlue;
checkBox.CheckedForeColor = Color.DarkBlue;
checkBox.UncheckedBackColor = Color.White;
checkBox.UncheckedForeColor = Color.Black;

Theme-Responsive States

  • Normal State - Uses theme's default checkbox colors
  • Hover State - Slightly highlighted appearance
  • Focused State - Visual focus indicator for keyboard navigation
  • Checked State - Theme's checked appearance with checkmark
  • Indeterminate State - Special styling for partial selection
  • Disabled State - Grayed out appearance
üé® Theming Tip

Use consistent checkbox styling across your application by applying the same theme to all BeepCheckBox instances for a unified look.

Best Practices

üé® User Experience

  • Clear labeling: Use descriptive text that clearly explains what the checkbox controls
  • Logical grouping: Group related checkboxes together visually
  • Appropriate spacing: Provide adequate spacing between checkboxes for easy clicking
  • Consistent alignment: Align checkboxes consistently (usually left-aligned)
  • Three-state usage: Only use indeterminate state when it adds value (like "Select All" scenarios)

‚ö° Performance

  • Event handling: Avoid heavy processing in CheckedChanged events
  • Batch updates: When updating multiple checkboxes, temporarily disable events
  • Efficient validation: Validate form state only when necessary
  • Memory management: Properly dispose of event handlers in long-running applications

‚ôø Accessibility

  • Keyboard navigation: Ensure proper tab order for keyboard users
  • Screen readers: Use meaningful text that describes the checkbox function
  • High contrast: Test with high contrast themes for visibility
  • Focus indicators: Ensure focused checkboxes have clear visual indicators
  • Touch targets: Make clickable areas large enough for touch interfaces

üìã Data Binding

  • Property binding: Use the Checked property for simple boolean binding
  • State binding: Use CheckState property when supporting three states
  • Validation: Implement proper validation for required checkboxes
  • Change tracking: Track changes for undo/redo functionality
‚ö†Ô∏è State Management

When using three-state checkboxes, be careful about how user interactions affect the indeterminate state. Users can't directly set indeterminate state through clicking.

‚úÖ Pro Tip

Use CheckStateChanged instead of CheckedChanged when working with three-state checkboxes to capture all state transitions including indeterminate states.
