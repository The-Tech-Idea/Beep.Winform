Overview

BeepCircularButton is a modern, circular button control perfect for floating action buttons (FABs), icon buttons, or anywhere a circular design aesthetic is needed. It provides Material Design-inspired appearance with customizable sizes and animations.

Built on the BeepControl foundation, BeepCircularButton features perfect circular geometry, icon integration, hover effects, ripple animations, size variants, and comprehensive theming support. Perfect for primary actions, floating buttons, icon launchers, and modern interface elements.

Basic Usage

Simple Circular Button

// Create a basic circular button
var circularButton = new BeepCircularButton();
circularButton.Size = new Size(48, 48);
circularButton.BackColor = Color.FromArgb(25, 118, 210);
circularButton.ForeColor = Color.White;
circularButton.Text = "+";
circularButton.Font = new Font("Segoe UI", 16, FontStyle.Bold);

// Add click handler
circularButton.Click += (s, e) =>
{
    MessageBox.Show("Circular button clicked!");
};

// Add to form
this.Controls.Add(circularButton);

Floating Action Button (FAB)

// Create a floating action button
var fabButton = new BeepCircularButton
{
    Size = new Size(64, 64),
    BackColor = Color.FromArgb(244, 67, 54), // Material Red
    ForeColor = Color.White,
    Text = "‚úâ", // Email icon
    Font = new Font("Segoe UI", 18),
    Anchor = AnchorStyles.Bottom | AnchorStyles.Right,
    Location = new Point(this.Width - 80, this.Height - 80),
    EnableRippleEffect = true,
    ShowShadow = true
};

// Handle click to open compose dialog
fabButton.Click += (s, e) =>
{
    OpenComposeEmailDialog();
};

this.Controls.Add(fabButton);

Icon Button with Image

// Create icon button with image
var iconButton = new BeepCircularButton
{
    Size = new Size(40, 40),
    BackColor = Color.White,
    ForeColor = Color.FromArgb(100, 100, 100),
    Image = Properties.Resources.SettingsIcon, // Load from resources
    ImageSize = new Size(20, 20),
    BorderStyle = BorderStyle.FixedSingle,
    FlatStyle = FlatStyle.Flat
};

// Remove border for clean look
iconButton.FlatAppearance.BorderSize = 1;
iconButton.FlatAppearance.BorderColor = Color.FromArgb(200, 200, 200);

// Handle settings action
iconButton.Click += (s, e) =>
{
    ShowSettingsDialog();
};

this.Controls.Add(iconButton);

Theming

BeepCircularButton provides comprehensive theming support with automatic application of theme colors and effects:

Theme Application

var circularButton = new BeepCircularButton
{
    Theme = "DarkTheme",
    UseThemeColors = true,
    CircularSize = CircularButtonSize.Large
};

// Theme automatically applies:
// - BackColor: Primary theme color
// - ForeColor: Contrasting text color
// - HoverColor: Lighter/darker variant
// - ShadowColor: Appropriate shadow color

// Apply theme manually if needed
circularButton.ApplyTheme();

Button-Specific Theming

  • Color variants - Primary, secondary, success, warning, danger themes
  • Shadow effects - Theme-appropriate shadow colors and intensities
  • Hover states - Automatic hover color calculation
  • Ripple effects - Theme-aware ripple colors
