Overview

BeepTextBox is an enhanced text input control that extends the standard Windows Forms TextBox with Material Design principles, built-in validation, floating labels, placeholder text, and comprehensive theming support.

Built on the BeepControl foundation, BeepTextBox provides a modern user experience with features like input validation, character counting, password visibility toggles, and seamless integration with the Beep theming system.

Basic Usage

Simple Text Input

// Create a basic text box
var textBox = new BeepTextBox
{
    PlaceholderText = "Enter your name",
    Size = new Size(250, 30),
    Location = new Point(10, 10)
};

this.Controls.Add(textBox);

Text Box with Floating Label

// Create text box with floating label
var emailTextBox = new BeepTextBox
{
    PlaceholderText = "Enter your email",
    FloatingLabelText = "Email Address",
    HasFloatingLabel = true,
    ValidationType = InputValidationType.Email,
    Size = new Size(300, 50),
    Location = new Point(10, 70)
};

this.Controls.Add(emailTextBox);

Password Input

// Create password input with visibility toggle
var passwordTextBox = new BeepTextBox
{

    UseSystemPasswordChar = true,

Validation

BeepTextBox includes built-in validation with visual feedback and error messages.

Validation Types

Type Description Example
None No validation Any text input
Email Valid email format user@example.com
URL Valid URL format https://example.com
Number Numeric input only 123.45
Custom Custom regex pattern User-defined pattern

Validation Examples

// Email validation
var emailField = new BeepTextBox
{
    PlaceholderText = "Enter email address",
    ValidationType = InputValidationType.Email,
    IsRequired = true,
    ErrorMessage = "Please enter a valid email address"
};

// Custom validation (phone number)
var phoneField = new BeepTextBox
{
    PlaceholderText = "Enter phone number",
    ValidationType = InputValidationType.Custom,
    ValidationPattern = @"^\+?[\d\s\-\(\)]{10,}$",
    ErrorMessage = "Please enter a valid phone number"
};

// Number validation with range
var ageField = new BeepTextBox
{
    PlaceholderText = "Enter your age",
    ValidationType = InputValidationType.Number,
    MinValue = 18,
    MaxValue = 120,
    ErrorMessage = "Age must be between 18 and 120"
};

// Validate manually
if (emailField.ValidateInput())
{
    // Input is valid
    ProcessEmail(emailField.Text);
}
else
{
    // Show validation errors
    ShowErrorMessage(emailField.ErrorMessage);
}
üìù Note:

Validation occurs automatically on TextChanged and Leave events. You can also trigger validation manually using the ValidateInput() method.

Examples

Registration Form

private void CreateRegistrationForm()
{
    var panel = new BeepPanel
    {
        Dock = DockStyle.Fill,
        ShowTitle = true,
        TitleText = "User Registration",
        Padding = new Padding(20)
    };

    // Username field
    var usernameField = new BeepTextBox
    {
        PlaceholderText = "Choose a username",
        FloatingLabelText = "Username",
        HasFloatingLabel = true,
        IsRequired = true,
        MinLength = 3,
        MaxLength = 20,
        Size = new Size(300, 50),
        Location = new Point(20, 60)
    };

    // Email field
    var emailField = new BeepTextBox
    {
        PlaceholderText = "Enter your email",
        FloatingLabelText = "Email Address",
        HasFloatingLabel = true,
        ValidationType = InputValidationType.Email,
        IsRequired = true,
        Size = new Size(300, 50),
        Location = new Point(20, 120)
    };

    // Password field
    var passwordField = new BeepTextBox
    {
        PlaceholderText = "Create a password",
        FloatingLabelText = "Password",
        HasFloatingLabel = true,
        UseSystemPasswordChar = true,
        ShowPasswordToggle = true,
        MinLength = 8,
        IsRequired = true,
        Size = new Size(300, 50),
        Location = new Point(20, 180)
    };

    // Confirm password field
    var confirmPasswordField = new BeepTextBox
    {
        PlaceholderText = "Confirm your password",
        FloatingLabelText = "Confirm Password",
        HasFloatingLabel = true,
        UseSystemPasswordChar = true,
        IsRequired = true,
        Size = new Size(300, 50),
        Location = new Point(20, 240)
    };

    // Custom validation for password confirmation
    confirmPasswordField.TextChanged += (s, e) => {
        if (confirmPasswordField.Text != passwordField.Text)
        {
            confirmPasswordField.SetError("Passwords do not match");
        }
        else
        {
            confirmPasswordField.ClearError();
        }
    };

    panel.Controls.AddRange(new Control[] 
    { 
        usernameField, emailField, passwordField, confirmPasswordField 
    });

    this.Controls.Add(panel);
}

Search Form with Auto-Complete

private void CreateSearchForm()
{
    var searchBox = new BeepTextBox
    {
        PlaceholderText = "Search products, categories, or brands...",
        Size = new Size(400, 40),
        Location = new Point(20, 20),
        Font = new Font("Segoe UI", 12),
        ShowClearButton = true
    };

    // Add search icon
    searchBox.LeftIcon = "üîç";

    // Auto-complete functionality
    var autoCompleteSource = new AutoCompleteStringCollection();
    autoCompleteSource.AddRange(new string[] 
    {
        "Laptops", "Smartphones", "Tablets", "Headphones",
        "Keyboards", "Mice", "Monitors", "Speakers"
    });

    searchBox.AutoCompleteMode = AutoCompleteMode.SuggestAppend;
    searchBox.AutoCompleteSource = AutoCompleteSource.CustomSource;
    searchBox.AutoCompleteCustomSource = autoCompleteSource;

    // Search event
    searchBox.KeyDown += (s, e) => {
        if (e.KeyCode == Keys.Enter)
        {
            PerformSearch(searchBox.Text);
        }
    };

    this.Controls.Add(searchBox);
}

Multi-line Text Area

private void CreateCommentForm()
{
    var commentBox = new BeepTextBox
    {
        PlaceholderText = "Write your comment here...",
        FloatingLabelText = "Comment",
        HasFloatingLabel = true,
        Multiline = true,
        MaxLength = 500,
        ShowCharacterCount = true,
        ScrollBars = ScrollBars.Vertical,
        Size = new Size(400, 120),
        Location = new Point(20, 20)
    };

    // Auto-resize based on content
    commentBox.TextChanged += (s, e) => {
        var lines = commentBox.Lines.Length;
        var newHeight = Math.Max(120, Math.Min(300, lines * 20 + 40));
        if (commentBox.Height != newHeight)
        {
            commentBox.Height = newHeight;
        }
    };

    this.Controls.Add(commentBox);
}

Theming

BeepTextBox automatically adapts to your application's theme, changing colors, fonts, and visual effects.

Applying Themes

// Apply theme to individual text box
textBox.Theme = "DarkTheme";
textBox.ApplyTheme();

// Apply theme to all text boxes in a form
private void ApplyThemeToForm(string themeName)
{
    foreach (Control control in this.Controls)
    {
        if (control is BeepTextBox textBox)
        {
            textBox.Theme = themeName;
            textBox.ApplyTheme();
        }
    }
}

Theme-Affected Properties

  • Background and border colors - Adapt to theme color scheme
  • Text and placeholder colors - Use theme foreground colors
  • Focus and validation colors - Use theme accent colors
  • Font family and sizing - Apply theme typography settings

Best Practices

üé® User Experience

  • Use meaningful placeholders: Provide examples or hints about expected input format
  • Enable floating labels: For better space utilization and visual hierarchy
  • Provide immediate feedback: Use validation colors and messages for real-time feedback
  • Group related fields: Use consistent spacing and alignment

‚ö° Performance

  • Debounce validation: Don't validate on every keystroke for expensive operations
  • Use appropriate validation types: Built-in validation is faster than custom regex
  • Limit auto-complete items: Too many items can slow down the UI

‚ôø Accessibility

  • Provide clear labels: Use floating labels or associated label controls
  • Error announcements: Ensure error messages are accessible to screen readers
  • Tab order: Set appropriate tab indices for keyboard navigation
  • High contrast support: Test with high contrast themes
‚ö†Ô∏è Common Pitfalls

Avoid using placeholder text as the only label - it disappears when users start typing, making it difficult to remember what the field is for.

‚úÖ Pro Tip

Use the ShowCharacterCount property for fields with length limits to help users understand constraints before they hit them.
