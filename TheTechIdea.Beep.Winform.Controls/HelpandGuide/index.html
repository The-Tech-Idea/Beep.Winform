Beep Controls Documentation

Modern, themeable WinForms controls for .NET 6, 7, 8, and 9 applications

Overview

TheTechIdea.Beep.Winform.Controls is a comprehensive library of modern, themeable WinForms controls designed for .NET 6, 7, 8, and 9 applications. Built with React-style properties, Material Design principles, and advanced features like SVG support, animations, and sophisticated theming systems.

This library provides over 80+ controls ranging from basic input controls to complex data visualization components, all designed to work seamlessly together with consistent styling and behavior.

Key Features

🎨 Rich Theming System

Complete theme support with multiple built-in themes including dark mode, high contrast, and custom themes.

⚛️ React-Style Properties

Modern component variants, sizes, colors, and elevation properties similar to React UI libraries.

🎯 Material Design

Built-in Material Design components with ripple effects, floating labels, and proper elevation.

🖼️ SVG Support

Native SVG rendering with theme-aware coloring and scaling capabilities.

✨ Animations & Effects

Built-in animations, transitions, and visual effects for enhanced user experience.

🔧 Easy Integration

Simple to integrate into existing WinForms applications with minimal code changes.

Install the package using one of the following methods:

Package Manager Console

.NET CLI

dotnet add package TheTechIdea.Beep.Winform.Controls

Basic Usage

Here's a simple example to get you started:

Creating Your First Beep Control

using TheTechIdea.Beep.Winform.Controls;

public partial class Form1 : Form
{
    public Form1()
    {
        InitializeComponent();
        
        // Create a BeepButton
        var beepButton = new BeepButton
        {
            Text = "Hello Beep!",
            Size = new Size(150, 40),
            Location = new Point(50, 50),
            UIVariant = ReactUIVariant.Contained,
            UIColor = ReactUIColor.Primary,
            Theme = "DefaultTheme"
        };
        
        // Add click event
        beepButton.Click += (s, e) => MessageBox.Show("Hello from Beep Controls!");
        
        // Add to form
        this.Controls.Add(beepButton);
    }
}
