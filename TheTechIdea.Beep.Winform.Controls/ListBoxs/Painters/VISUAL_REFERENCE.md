## Color Conventions

### Primary Color (Selection)
```
Used for:
- Selection background (40-60% opacity)
- Selection border (100% opacity)
- Focus indicators
- Primary action highlights

Example RGB Values:
- Light Blue: (100, 180, 220)
- Accent Blue: (70, 150, 200)
```

### Accent Color (Hover)
```
Used for:
- Hover borders (1-1.5px)
- Secondary highlights
- Hover overlays (10-20% opacity)
- Alternative indicators

Example RGB Values:
- Light Gray: (100, 150, 200)
- Gray: (180, 180, 180)
```

### Background Colors
```
Used for:
- Normal item background
- Hover overlays
- Disabled states
- Secondary backgrounds

Example RGB Values:
- Off-white: (250, 250, 250)
- Light gray: (240, 240, 240)
- Border: (220, 220, 220)
```

## Shadow and Elevation Levels

### No Shadow
- Normal state for minimal painters
- Flat appearance

### Subtle Shadow (1px, 10-20% black)
- Normal state for card painters
- Slight elevation

### Medium Shadow (2px, 20-30% black)
- Hover state for elevated painters
- Increased depth perception

### Strong Shadow (3px, 30-50% black)
- Selected state for card painters
- Maximum elevation

### Gradient Shadow
- Linear gradient from darker to transparent
- 90° vertical direction
- Smooth elevation effect

## Accessibility Considerations

? **Color Contrast:**
- Text always readable on background
- Minimum 4.5:1 contrast ratio

? **Visual Indicators:**
- Don't rely on color alone
- Use borders, shadows, shapes
- Left bars, checkmarks, fills

? **Focus States:**
- Clear focus outline
- High contrast border
- Optional dotted/dashed style

? **Hover Feedback:**
- Clear visual change
- Smooth transition (if applicable)
- Obvious to all users

---

## Animation Considerations

While current implementation is static, borders/effects could be animated:
- **Transition Duration:** 150-250ms
- **Easing:** Ease-in-out

## Responsive Design

### High-DPI Scaling
- Borders scale with DPI
- Shadows adjust proportionally
- Radius values scale

### Small Screens
- CompactListPainter recommended (24px height)
- Borders still visible
- Touch targets adequate

### Large Displays
- All painters scale well
- Elevated painters recommended (CardList, Rounded)
- Shadows more pronounced
